# 《问道》游戏设计文档

> 类型：卡牌 Roguelike | 参考：杀戮尖塔 | 世界观：修仙（《剑来》风格）
> 设计日期：2026-05-01

---

## 一、整体架构

### 1.1 核心游戏循环

```
选角色 → 从山脚出发 → 每层境界（Act）选路线爬升
→ 每个节点是事件（战斗/机缘/坊市/闭关/天劫）
→ 击败境界 Boss（天劫）→ 突破 → 进入下一境界
→ 飞升成功 → 回到山脚（下一循环开始）
→ 死亡 → 回到山脚（肉鸽重置）
```

### 1.2 叙事设定：循环

玩家扮演一个困于循环中的修士。每一次跑图 = 一世修炼。飞升之后，发现自己又回到了起点。为什么困在循环里，是玩家在机缘事件中逐渐拼凑的谜题。

- **剑修循环理由：** 立下天道誓言，誓要斩破这方虚假天地
- **法修循环理由：** 窥见天道裂痕，知道飞升是假，真相在循环深处
- **体修循环理由：** 不关心为什么，只信"百世轮回，终有一身破天"

### 1.3 境界（Act）结构

| Act | 境界 | Boss |
|-----|------|------|
| 1 | 筑基 | 心魔劫 |
| 2 | 金丹 | 四九天劫 |
| 3 | 元婴 | 六九天劫 |
| 4 | 化神·飞升 | 九九天劫 |

### 1.4 节点类型

| 节点 | 内容 |
|------|------|
| 散修/妖兽 | 基础战斗，掉卡/灵石 |
| 邪修/大妖 | 精英战斗，掉稀有卡/遗物 |
| 坊市 | 花灵石买卡/遗物/丹药 |
| 闭关 | 回真气/回血，或悟道（升级）一张卡 |
| 机缘 | 随机叙事事件，可得遗物/卡/诅咒 |
| 天劫 | 境界 Boss，击败后突破 |

## 二、卡牌系统

### 2.1 基础框架

| 概念 | 设定 |
|------|------|
| 能量 | 真气（基础 3/回合） |
| 攻击牌 | 招式 |
| 技能牌 | 身法 |
| 能力牌 | 功法 |
| 升级 | 悟道（在闭关处执行） |
| 费用 | 0-3 真气 |

### 2.2 稀有度

| 品级 | 对应 STS | 掉落来源 |
|------|---------|---------|
| 凡品 | Common | 普通战斗 |
| 灵品 | Uncommon | 精英/坊市/机缘 |
| 仙品 | Rare | 精英/机缘/天劫 |

### 2.3 初始牌组（每个角色 12 张）

通用框架：5 张基础招式（1 费打 6）+ 5 张基础身法（1 费格挡 5）+ 1 张本命功法 + 1 张特色牌。

### 2.4 状态牌（负面）

| 牌名 | 效果 |
|------|------|
| 心魔 | 不能打出，回合结束在手中则失去 1 HP |
| 走火入魔 | 不能打出，回合结束在手中则随机丢弃 1 张牌 |
| 业火 | 回合结束在手中则受到 2 伤害 |

### 2.5 悟道（卡牌升级）

在闭关处可选一张卡悟道。升级效果优先改机制或减费，不堆数值。

### 3.2 遗物类型

| 类型 | 效果 |
|------|------|
| 法宝 | 被动常驻效果 |
| 丹药 | 一次性或条件触发 |
| 功法残卷 | 修改卡牌/真气规则 |

### 3.3 各角色本命法宝

| 角色 | 法宝 | 效果 |
|------|------|------|

| 体修 | 血玉 | 每场战斗首次受伤额外获得 2 层气血 |

### 3.4 渡劫遗宝（设计示例）

| 遗宝 | 效果 | 代价 |
|------|------|------|

## 四、地图系统

### 4.1 天梯图

每层境界为一张分层路线图，玩家从底向上选路。每条路线连接不同节点。结构严格分层，类似 STS，玩家每次可在同层节点中做选择，不能返回已过的层。

### 4.2 节点分布（每境界约 12-18 个节点）

| 节点类型 | 占比 |
|---------|------|
| 散修/妖兽 | 45% |
| 邪修/大妖 | 12% |
| 坊市 | 8% |
| 闭关 | 15% |
| 机缘 | 12% |
| 天劫 | 1 个（终点） |

### 4.3 随机化

每局地图随机生成节点布局和连接路线。高难度下，精英节点占比增大，休息节点减少。

## 五、怪物系统

### 5.1 意图系统

沿用 STS 透明意图：每个怪物头顶显示下回合动作（图标+数值），玩家据此决策。

### 5.2 普通怪物行为类型

| 类型 | 修仙设定 | 行为模式 |
|------|---------|---------|
| 攻击型 | 妖兽/散修 | 高伤害，低血量 |
| 防御型 | 护山灵兽/傀儡 | 上格挡，间歇攻击 |
| 干扰型 | 魅修/幻妖 | 塞状态牌，低伤害 |
| 爆发型 | 雷劫兽/火灵 | 蓄力多回合后爆发 |
| 群体型 | 小妖群/蛊虫 | 多只低血量同时出场 |
| 辅助型 | 妖将/阵旗 | 给队友 buff |

### 5.3 精英怪物

每境界 3 种精英，随机出 1 种。带被动特性。

**筑基境：**

| 精英 | 被动 | 机制 |
|------|------|------|
| 夺舍邪修 | 每 3 回合偷玩家 1 层正面 buff | 拖长盘消耗 |
| 三首妖蟒 | 每回合切换攻/防/毒模式 | 三形态轮转 |
| 血煞 | 杀小怪回血 15 | 带两只小怪 |

**金丹境：**

| 精英 | 被动 | 机制 |
|------|------|------|
| 金丹劫修 | 每 4 回合全屏雷劫 | 逼玩家抢攻 |
| 七情魔 | 每回合塞负面情绪牌 | 污染牌组 |
| 双生子剑修 | 共享血量，攻守分工，3 回合互换 | 考验切换目标 |

### 5.4 Boss（天劫）

| 境界 | Boss | 机制 |
|------|------|------|
| 筑基 | 心魔劫 | 复制玩家牌组中的牌作为攻击 |
| 金丹 | 四九天劫 | 36 雷分 4 波，波间可输出 |
| 元婴 | 六九天劫 | 天雷（单体）+ 天火（AOE）双部位 |
| 化神 | 九九天劫 | 三阶段：输出检验 → 污染 → 全面压榨 |

### 5.5 数值曲线

| 境界 | 普通怪 HP | 精英 HP | Boss HP |
|------|----------|---------|---------|
| 筑基 | 40-70 | 110-150 | 250-300 |
| 金丹 | 55-90 | 140-190 | 320-380 |
| 元婴 | 70-110 | 180-240 | 400-480 |
| 化神 | 90-140 | 220-300 | 500-600 |

## 六、玩家角色系统

### 6.1 剑修 — 剑意叠加

**核心机制：剑意**
- 打出「招式」牌获得 1 层剑意
- 剑意上限 10 层（养剑术后 15）
- 每层剑意使下张招式 +1 伤害
- 「终结技」牌消耗全部剑意产生爆发
- 回合结束时剑意保留一半（向下取整）

**玩法：** 攒剑意 → 身法防御 → 终结技爆发。

**本命法宝：** 养剑葫 — 每场战斗开始获得 3 层剑意

### 6.2 法修 — 五行轮转

**核心机制：五行位**
- 金→水→木→火→土→金（单向循环）
- 打出元素牌推动五行位
- 按"相生"顺序打出触发五行连珠：额外效果
- 当前位提供常驻增益：

| 位 | 效果 |
|----|------|
| 金 | 招式 +2 伤害 |
| 水 | 身法 +2 格挡 |
| 木 | 回合结束回 2 HP |
| 火 | 招式 +25% 伤害 |
| 土 | 回合开始 +1 临时真气 |

**玩法：** 规划牌序，连珠+站位最大化收益。

**本命法宝：** 五行盘 — 每场战斗首次切换五行位不耗真气

### 6.3 体修 — 气血转化

**核心机制：气血**
- 受伤（扣 HP）时获得 1 层气血（每回合最多 3 层）
- 打出「炼体」牌消耗气血换取收益
- 血量即资源，高风险高回报

**玩法：** 挨打攒气血 → 气血转输出或回血。

**本命法宝：** 血玉 — 每场战斗首次受伤额外获得 2 层气血

### 6.4 解锁顺序

| 顺序 | 角色 | 条件 | 难度 |
|------|------|------|------|
| 1 | 剑修 | 直接可用 | 低 |
| 2 | 法修 | 剑修通关一次 | 中 |
| 3 | 体修 | 法修通关一次 | 高 |

## 七、跨局成长

### 7.1 轮回碎片（叙事收集）

在机缘事件中可收集轮回碎片。碎片不增加数值强度，只推进真相揭露。类似 STS 的角色解锁记忆/钥匙/红宝石。

### 7.2 角色解锁

见 6.4。后续可扩展更多角色。

### 7.3 难度递增（飞升模式）

类似 STS 飞升等级（Ascension），每次通关后解锁更高难度，敌方数值提升，负面规则叠加。

## 八、经济系统

| 资源 | 说明 |
|------|------|
| 灵石 | 在坊市消费，战斗掉落/机缘获取 |

| 真气 | 每回合战斗内刷新，基础 3/回合 |
