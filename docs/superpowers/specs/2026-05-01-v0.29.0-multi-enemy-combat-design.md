# v0.29.0 多敌人战斗 Design Spec

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 支持一场战斗中最多出现 6 个敌人；玩家通过点选目标的方式对单个敌人打出攻击牌，敌人按顺序依次行动。

**Architecture:** 纯扩展现有结构。新增 `EnemyGroupData` 资源；`CardData` 新增 `target_type` 字段；`CombatEngine` 将单个 `enemy` 改为 `enemies: Array[Combatant]`，`play_card` 接受 `target_index`；`CombatScreen` 动态生成敌人面板并处理目标选择交互；`NodeConfig` / `MapGenerator` 切换到 group 粒度。

**Tech Stack:** Godot 4 GDScript，数据层 `.tres` 文件。

---

## Section 1：数据层

### CardData.gd

新增导出字段：

```gdscript
@export var target_type: String = "single"
```

取值说明：
- `"single"` — 选一个敌人（默认值，所有攻击/减益牌）
- `"none"` — 仅影响玩家，无需选敌（defend / energize / insight / venom）

现有卡牌需要修改的 `.tres`（其余保持默认 `"single"`）：
- `data/cards/defend.tres` → `target_type = "none"`
- `data/cards/energize.tres` → `target_type = "none"`
- `data/cards/insight.tres` → `target_type = "none"`
- `data/cards/venom.tres` → `target_type = "none"`

### EnemyGroupData.gd

新建资源类，路径 `res://EnemyGroupData.gd`：

```gdscript
class_name EnemyGroupData
extends Resource

@export var enemies: Array[EnemyData] = []
```

### data/enemy_groups/*.tres

新建目录 `data/enemy_groups/`，包含以下 6 个分组文件：

| 文件 | enemies 数组 | 用于列数 |
|------|-------------|----------|
| `single_jaw_worm.tres` | [jaw_worm] | 早期 col 0–3 |
| `single_fire_lizard.tres` | [fire_lizard] | 早期 col 0–3 |
| `pair_early.tres` | [jaw_worm, fire_lizard] | 早期 col 0–3 |
| `single_poison_spider.tres` | [poison_spider] | 中期 col 4+ |
| `single_swamp_slime.tres` | [swamp_slime] | 中期 col 4+ |
| `pair_mid.tres` | [poison_spider, swamp_slime] | 中期 col 4+ |

## Section 2：CombatEngine 重构

### 字段变更

```gdscript
# 删除
var enemy: Combatant
var _enemy_data: EnemyData

# 新增
var enemies: Array[Combatant]
var _enemy_data_list: Array[EnemyData]
```

### setup() 签名变更

```gdscript
func setup(initial_deck: Array, enemy_group: EnemyGroupData,
           initial_hp: int, max_hp: int, relics: Array) -> void:
    # ...原有玩家初始化不变...
    for data in enemy_group.enemies:
        var c := Combatant.new()
        c.display_name = data.display_name
        c.hp = data.hp
        c.max_hp = data.hp
        enemies.append(c)
        _enemy_data_list.append(data)
    _start_player_turn()
```

### play_card() 签名变更

```gdscript
func play_card(card_index: int, target_index: int) -> bool:
    # target_index = -1 → 不需要敌人目标（target_type == "none" 的牌）
    # target_index >= 0 → enemies[target_index]
    var card: CardData = hand[card_index]
    if card.cost > energy:
        return false
    energy -= card.cost
    hand.remove_at(card_index)
    _discard_pile.append(card)
    if target_index >= 0:
        EffectResolver.resolve(card, player, enemies[target_index])
    else:
        EffectResolver.resolve(card, player, null)
    state_changed.emit()
    _check_end()
    return true
```

> **注意**：`EffectResolver.resolve` 当 `target` 为 null 时只执行玩家侧效果（heal / block / draw / energy）。需确认 EffectResolver 已能处理 null target（当前代码应已能处理，因为玩家效果不访问 target）。

### 新增辅助方法

```gdscript
func _get_living_enemies() -> Array[Combatant]:
    return enemies.filter(func(e: Combatant) -> bool: return e.hp > 0)

func get_enemy_action(i: int) -> EnemyActionData:
    var actions: Array = _enemy_data_list[i].actions
    return actions[turn_number % actions.size()]
```

删除旧方法 `get_current_enemy_action()`。

### _do_enemy_turn() 改为循环

```gdscript
func _do_enemy_turn() -> void:
    for i in enemies.size():
        if enemies[i].hp <= 0:
            continue
        enemies[i].block = 0
        enemies[i].vulnerable = max(0, enemies[i].vulnerable - 1)
        var action: EnemyActionData = get_enemy_action(i)
        if action.type == "attack":
            EffectResolver.apply_damage(enemies[i], player, action.value)
            enemies[i].weak = max(0, enemies[i].weak - 1)
        elif action.type == "poison":
            var venom_card: CardData = load("res://data/cards/venom.tres") as CardData
            for j in action.value:
                _draw_pile.append(venom_card.duplicate())
            _draw_pile.shuffle()
        else:
            enemies[i].add_block(action.value)
```

`_do_enemy_turn()` 自身不调用 `_start_player_turn()`——这一步由 `end_turn()` 在 `state_changed.emit()` 之后负责，保持与现有结构一致。

### 胜利条件

`_check_end()` 中：存活敌人数为 0 时触发 `combat_ended.emit("win")`。

## Section 3：CombatScreen UI 重构

### 场景结构变更（CombatScreen.tscn）

删除原有单个 `EnemyPanel` 节点，在战斗区域加入 `HBoxContainer`（命名 `EnemiesContainer`）。面板宽度由 Godot 自适应分配（敌人少时宽，多时窄）。

### 动态生成敌人面板

```gdscript
@onready var _enemies_container: HBoxContainer = $EnemiesContainer

func _build_enemy_panels() -> void:
    for child in _enemies_container.get_children():
        child.queue_free()
    for i in _engine.enemies.size():
        var panel := _make_enemy_panel(i)
        _enemies_container.add_child(panel)

func _make_enemy_panel(i: int) -> Control:
    var btn := Button.new()
    btn.pressed.connect(_on_enemy_pressed.bind(i))
    # 面板内容在 _refresh_ui() 中更新
    return btn
```

### _refresh_ui() 更新敌人面板

```gdscript
func _refresh_ui() -> void:
    # ...玩家 HP / 手牌不变...
    for i in _engine.enemies.size():
        var e: Combatant = _engine.enemies[i]
        var action: EnemyActionData = _engine.get_enemy_action(i)
        var panel: Button = _enemies_container.get_child(i)
        panel.text = "%s\nHP:%d  Block:%d\n意图:%s(%d)" % [
            e.display_name, e.hp, e.block, action.type, action.value
        ]
        panel.disabled = e.hp <= 0
```

### 目标选择流程

```gdscript
var _pending_card_index: int = -1

func _on_card_pressed(card_index: int) -> void:
    var card: CardData = _engine.hand[card_index]
    if card.target_type == "none":
        _engine.play_card(card_index, -1)
    else:  # "single"
        _pending_card_index = card_index
        _set_targeting_mode(true)

func _on_enemy_pressed(enemy_index: int) -> void:
    if _pending_card_index >= 0:
        _engine.play_card(_pending_card_index, enemy_index)
        _pending_card_index = -1
        _set_targeting_mode(false)

func _set_targeting_mode(active: bool) -> void:
    # active=true：高亮敌人按钮，禁用手牌按钮
    # active=false：恢复正常状态
    for btn in _hand_buttons:
        btn.disabled = active
    for i in _enemies_container.get_child_count():
        var panel: Button = _enemies_container.get_child(i)
        if _engine.enemies[i].hp > 0:
            panel.disabled = not active
```

取消选择：选敌模式下再次点击同一张手牌，清除 `_pending_card_index` 并退出高亮。
