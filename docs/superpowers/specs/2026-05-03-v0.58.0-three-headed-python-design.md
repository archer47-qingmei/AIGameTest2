# 三首妖蟒 实现设计 (v0.58.0)

## 目标

新增第二只精英怪：三首妖蟒，引入 phase2 HP 阈值机制和 `poison_curse` 复合动作类型。

---

## 怪物数据

| 属性 | 值 |
|------|-----|
| HP | 147 |
| 行动模式 | 顺序3回合循环 |
| 攻首（phase 1 & 2） | 攻击 13 |
| 防首（phase 1 & 2） | 格挡 14 |
| 毒首 phase 1（HP ≥ 50%） | 投毒 4层 |
| 毒首 phase 2（HP < 50%） | 投毒 4层 + 心魔 ×2 |

- 阈值：`phase2_threshold = 0.5`（HP 降至 50% 及以下后永久切换至 phase 2 actions）
- 顺序计数跨 phase 切换不重置（第几回合就还是第几位）

---

## 架构改动

### 1. EnemyData.gd

新增两个导出字段：

```gdscript
@export var phase2_threshold: float = 0.0
@export var phase2_actions: Array[EnemyActionData] = []
```

- `phase2_threshold = 0.0` 表示不启用 phase2（所有现有怪物默认值，不影响现有行为）
- `phase2_actions` 为空时同样不启用

### 2. CombatEngine.gd

**`_resolve_enemy_actions()` 改动：**

在取 action 时，判断是否进入 phase2：

```gdscript
var data: EnemyData = _enemy_data_list[i]
var use_phase2: bool = (
    data.phase2_threshold > 0.0
    and not data.phase2_actions.is_empty()
    and float(enemies[i].hp) / float(data.hp) <= data.phase2_threshold
)
var action_list: Array[EnemyActionData] = data.phase2_actions if use_phase2 else data.actions
var action: EnemyActionData = action_list[(turn_number - 1) % action_list.size()]
```

random_actions 模式不受 phase2 影响（三首妖蟒使用顺序模式）。

**`_do_enemy_turn()` 新增 `poison_curse` 分支：**

```gdscript
"poison_curse":
    for _j in action.value:
        _draw_pile.append(VENOM_CARD.duplicate())
    _draw_pile.shuffle()
    for _j in action.count:
        _discard_pile.append(VENOM_CARD.duplicate())
```

（value = 投入抽牌堆的毒层数，count = 投入弃牌堆的心魔数）

### 3. CombatScreen.gd — `_intent_text()`

新增 `poison_curse` 分支：

```gdscript
"poison_curse": return "意图：投毒 %d + 心魔 ×%d" % [action.value, action.count]
```

### 4. MapGenerator.gd

将 `ELITE_GROUP_PATH: String` 改为 `ELITE_GROUP_PATHS: Array[String]`（与 EARLY/MID 命名一致），`_make_column()` 随机选一个：

```gdscript
const ELITE_GROUP_PATHS: Array[String] = [
    "res://data/enemy_groups/single_elite_sorcerer.tres",
    "res://data/enemy_groups/single_three_headed_python.tres",
]
# _make_column() 中：
nd.config.enemy_group = load(ELITE_GROUP_PATHS[randi() % ELITE_GROUP_PATHS.size()]) as EnemyGroupData
```

同时更新 `TestSelectScreen.gd`：
```gdscript
# 旧：
all_paths.append(MapGenerator.ELITE_GROUP_PATH)
# 新：
all_paths.append_array(MapGenerator.ELITE_GROUP_PATHS)
```
