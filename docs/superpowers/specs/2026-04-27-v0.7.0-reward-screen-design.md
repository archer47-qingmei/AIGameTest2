# v0.7.0 战斗后奖励选牌设计文档

**日期**：2026-04-27  
**项目**：AIGameTest2（Godot 4.6 + GDScript）  
**范围**：v0.7.0 — 战斗胜利后进入奖励场景，从随机 3 张卡牌中选 1 张加入本局牌库

---

## 背景

v0.6.0 建立了完整游戏循环（主菜单 → 战斗 → 主菜单）。v0.7.0 在胜利路径上插入奖励场景，并引入 `PlayerState` 持有本局牌库状态，为 v0.8.0 多关卡地图打好基础。

---

## 架构原则遵守情况

- **PlayerState**：RefCounted 子类，纯数据，只持有牌库列表，无 Node 引用
- **RewardEngine**：RefCounted 静态方法，纯逻辑，不含 UI
- **RewardScreen**：Control UI 层，只做显示和事件转发，调用 RewardEngine 和 GameManager
- **GameManager**：持有 PlayerState 引用（基础设施层职责），不含游戏规则逻辑

## 模块设计

### PlayerState.gd（新增）

```gdscript
class_name PlayerState
extends RefCounted

var deck: Array[CardData] = []
```

职责：持有本局运行时的牌库组成。由 `GameManager.go_to_combat()` 创建并填入初始 10 张牌，奖励选牌后追加 1 张，`go_to_menu()` 时置 null 销毁。

---

### RewardEngine.gd（新增）

```gdscript
class_name RewardEngine
extends RefCounted

static func get_options() -> Array[CardData]:
    var options: Array[CardData] = []
    var dir := DirAccess.open("res://data/cards/")
    if dir == null:
        return options
    dir.list_dir_begin()
    var file_name := dir.get_next()
    while file_name != "":
        if file_name.ends_with(".tres"):
            var card := load("res://data/cards/" + file_name) as CardData
            if card != null:
                options.append(card)
        file_name = dir.get_next()
    options.shuffle()
    return options.slice(0, min(3, options.size()))
```

职责：枚举 `data/cards/` 下全部 `.tres` 文件，加载为 `CardData`，随机返回 3 张作为奖励选项。新增卡牌文件自动进入奖励池，无需额外配置。

### reward/RewardScreen.gd（新增）

```gdscript
extends Control

func _ready() -> void:
    var options: Array[CardData] = RewardEngine.get_options()
    for card: CardData in options:
        var btn := Button.new()
        var dmg: int = 0
        var blk: int = 0
        for effect: CardEffectData in card.effects:
            if effect.type == "damage":
                dmg = effect.value
            elif effect.type == "block":
                blk = effect.value
        btn.text = "%s\n费用:%d  攻:%d  挡:%d" % [card.card_name, card.cost, dmg, blk]
        btn.pressed.connect(_on_card_selected.bind(card))
        $CardContainer.add_child(btn)

func _on_card_selected(card: CardData) -> void:
    GameManager.player_state.deck.append(card.duplicate())
    GameManager.go_to_menu()
```

场景结构：Control 根节点 + Label（"选择一张卡牌加入牌库"）+ HBoxContainer（CardContainer，动态填入按钮）。

### GameManager.gd（修改）

新增 Phase.REWARD 枚举值，新增 `player_state` 字段，修改 `go_to_combat()` 初始化 PlayerState，新增 `go_to_reward()`，修改 `go_to_menu()` 清除 player_state。

```gdscript
extends Node

enum Phase { MENU, COMBAT, REWARD }

var current_phase: Phase = Phase.MENU
var player_state: PlayerState

func go_to_combat() -> void:
    player_state = PlayerState.new()
    var strike: CardData = preload("res://data/cards/strike.tres")
    var defend: CardData = preload("res://data/cards/defend.tres")
    var bash: CardData = preload("res://data/cards/bash.tres")
    var slash: CardData = preload("res://data/cards/slash.tres")
    for i in 4:
        player_state.deck.append(strike.duplicate())
    for i in 4:
        player_state.deck.append(defend.duplicate())
    player_state.deck.append(bash.duplicate())
    player_state.deck.append(slash.duplicate())
    current_phase = Phase.COMBAT
    get_tree().change_scene_to_file("res://combat/CombatScreen.tscn")

func go_to_reward() -> void:
    current_phase = Phase.REWARD
    get_tree().change_scene_to_file("res://reward/RewardScreen.tscn")

func go_to_menu() -> void:
    player_state = null
    current_phase = Phase.MENU
    get_tree().change_scene_to_file("res://menu/MainMenu.tscn")
```

### CombatEngine.gd（修改）

`setup()` 签名改为接受外部牌库，移除硬编码卡牌加载：

```gdscript
func setup(initial_deck: Array[CardData]) -> void:
    _enemy_data = preload("res://data/enemies/jaw_worm.tres")
    enemy = Combatant.new()
    enemy.display_name = _enemy_data.display_name
    enemy.hp = _enemy_data.hp
    enemy.max_hp = _enemy_data.hp
    enemy.block = 0

    player = Combatant.new()
    player.display_name = "玩家"
    player.hp = 70
    player.max_hp = 70
    player.block = 0

    for card: CardData in initial_deck:
        _draw_pile.append(card.duplicate())
    _draw_pile.shuffle()
    _start_player_turn()
```

### combat/CombatScreen.gd（修改）

`_ready()` 传入牌库，`_on_combat_ended()` 按胜负显示不同按钮：

```gdscript
@onready var _btn_get_reward: Button = $BtnGetReward    # 新增

func _ready() -> void:
    _engine = CombatEngine.new()
    _engine.state_changed.connect(_refresh_ui)
    _engine.combat_ended.connect(_on_combat_ended)
    _btn_end_turn.pressed.connect(_engine.end_turn)
    _btn_return_menu.pressed.connect(GameManager.go_to_menu)
    _btn_get_reward.pressed.connect(GameManager.go_to_reward)  # 新增
    _engine.setup(GameManager.player_state.deck)               # 修改

func _on_combat_ended(result: String) -> void:
    _lbl_result.text = "胜利！" if result == "victory" else "游戏结束"
    _lbl_result.visible = true
    _btn_end_turn.disabled = true
    if result == "victory":
        _btn_get_reward.visible = true     # 新增
    else:
        _btn_return_menu.visible = true
    for btn: Button in _hand_buttons:
        btn.disabled = true
```

`CombatScreen.tscn` 新增 `BtnGetReward` 节点（Button，visible=false，text="领取奖励"，锚点居中偏下，位于 BtnReturnMenu 旁边）。

## 游戏流程

```
MainMenu
  └─→ 点击"开始游戏" → GameManager.go_to_combat()
        创建 PlayerState，deck = 初始 10 张牌
        └─→ CombatScreen（setup 传入 PlayerState.deck）
              ├─→ 胜利 → 显示"胜利！"+ "领取奖励"按钮
              │         → GameManager.go_to_reward()
              │               └─→ RewardScreen
              │                     RewardEngine 随机 3 张
              │                     玩家选 1 张 → 加入 PlayerState.deck
              │                     → GameManager.go_to_menu()
              │                           player_state = null
              │                           └─→ MainMenu
              │
              └─→ 败北 → 显示"游戏结束"+ "返回主菜单"按钮
                        → GameManager.go_to_menu()
                              player_state = null
                              └─→ MainMenu
```
