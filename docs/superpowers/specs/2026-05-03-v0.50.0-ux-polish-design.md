# v0.50.0 UX 四项优化 Design

**Goal:** 修复地图初始定位、多段攻击动画、UI 布局，并新增拖牌伤害预览数字。

**Architecture:** 四个独立改动，互不依赖。Feature 4（伤害预览）通过新增 `EffectResolver.preview_damage()` 纯计算方法保持架构分层，DragLayer 只做展示，CombatScreen 负责计算传值。

**Tech Stack:** Godot 4.4 GDScript 严格类型，Tween chain，CanvasItem._draw()。

---

## Feature 1：地图初始定位

**文件：** `map/MapScreen.gd`

当前代码在 `await get_tree().process_frame` 后将 `scroll_vertical` 设为 `_map_content.get_minimum_size().y`（固定值 1650），导致不能正确反映玩家当前进度位置。

**改法：** 替换滚动目标为玩家 `available_nodes` 中第一个节点的 Y 坐标，使其居中显示：

```gdscript
await get_tree().process_frame
var target_y := _map_content.get_minimum_size().y  # 无可用节点时保底滚到底部
for nd: NodeData in _state.available_nodes:
    if _node_positions.has(nd):
        target_y = _node_positions[nd].y
        break
_scroll.scroll_vertical = int(target_y - _scroll.size.y / 2.0)
```

`_node_positions` 已在 `_build_map()` 中填充，`await` 后读取安全。

## Feature 2：多段攻击动画

**文件：** `EffectResolver.gd`、`CombatEngine.gd`、`combat/CombatScreen.gd`

### EffectResolver.resolve() 改返回 Array[int]

每次执行 "damage" 命中前后对比 `target.hp`，记录每段实际扣血量（已扣除格挡后的 HP 损失），拼入返回数组：

```gdscript
static func resolve(card: CardData, attacker: Combatant, defender: Combatant) -> Array[int]:
    var hit_amounts: Array[int] = []
    for effect: CardEffectData in card.effects:
        for _i in effect.times:
            match effect.type:
                "damage":
                    if defender != null:
                        var hp_before := defender.hp
                        # ... （原有 dmg_bonus 计算不变）
                        apply_damage(attacker, defender, effect.value + dmg_bonus)
                        hit_amounts.append(hp_before - defender.hp)
                # block/weak/vulnerable 等原有逻辑不变
    # is_finisher / 招式 后处理逻辑不变
    return hit_amounts
```

### CombatEngine 信号变更

删除旧信号 `damage_dealt(enemy_index: int, amount: int)`，新增：

```gdscript
signal hits_dealt(enemy_index: int, amounts: Array[int])
```

`play_card()` 中，将原有 `hp_before` 对比替换为直接用 `resolve()` 的返回值：

```gdscript
if card.target_type == "all":
    for i in enemies.size():
        if enemies[i].hp > 0:
            var amounts := EffectResolver.resolve(card, player, enemies[i])
            if not amounts.is_empty():
                hits_dealt.emit(i, amounts)
else:
    var target := enemies[target_index] if target_index >= 0 else null
    var amounts := EffectResolver.resolve(card, player, target)
    if target_index >= 0 and not amounts.is_empty():
        hits_dealt.emit(target_index, amounts)
```

### CombatScreen 顺序动画

将 `_engine.damage_dealt.connect(_on_damage_dealt)` 改为 `_engine.hits_dealt.connect(_on_hits_dealt)`。

新函数 `_on_hits_dealt(enemy_index: int, amounts: Array[int])` 用 Tween chain 顺序播放：

```gdscript

    for amount in amounts:
        if amount > 0:

            tw.tween_interval(0.22)

    # panel.pivot_offset、flash_tween、shake_tween、浮动数字 Label 不变
```

每段动画 0.22s 间隔，3 段总时长约 0.65s，与单次动画相近。`amount == 0`（完全被格挡）的命中不播放动画。

## Feature 4：拖牌伤害预览

**文件：** `EffectResolver.gd`、`combat/CombatScreen.gd`、`combat/DragLayer.gd`

### EffectResolver.preview_damage()

新增纯计算静态方法，不改变任何 Combatant 状态：

```gdscript
static func preview_damage(card: CardData, attacker: Combatant, target: Combatant) -> Dictionary:
    # 找第一个 "damage" 类型的 effect
    for effect: CardEffectData in card.effects:
        if effect.type == "damage":
            var dmg_bonus: int = 0
            if card.is_finisher:
                dmg_bonus = attacker.sword_intent * card.sword_intent_consume_bonus
            elif card.card_type == "招式":
                dmg_bonus = attacker.sword_intent * attacker.sword_intent_damage_bonus
            var base: int = effect.value + dmg_bonus
            var dmg: int = base
            if attacker.weak > 0:
                dmg = int(dmg * 0.75)
            if target.vulnerable > 0:
                dmg = int(dmg * 1.5)
            var boosted: bool = (target.vulnerable > 0) or (dmg_bonus > 0)  # 脆弱或剑意加成
            return {per_hit = dmg, hits = effect.times, boosted = boosted}
    return {}  # 无伤害效果（如纯防御牌）
```

### CombatScreen 计算预览值

在 `_on_card_button_down()` 中，构建 `enemy_positions` / `enemy_indices` 的同时，计算预览数据：

```gdscript
var damage_labels: Array[String] = []
var damage_boosted: Array[bool] = []

for i in _engine.enemies.size():
    if _engine.enemies[i].hp > 0:
        var preview := EffectResolver.preview_damage(card, _engine.player, _engine.enemies[i])
        if preview.is_empty():
            damage_labels.append("")
            damage_boosted.append(false)
        elif preview.hits == 1:
            damage_labels.append(str(preview.per_hit))
            damage_boosted.append(preview.boosted)
        else:
            damage_labels.append("%d×%d" % [preview.per_hit, preview.hits])
            damage_boosted.append(preview.boosted)
```

将 `damage_labels` 和 `damage_boosted` 作为第八、九个参数传入 `begin_drag()`。

### DragLayer 接口与渲染

**新增字段：**
```gdscript
var _damage_labels: Array[String] = []
var _damage_boosted: Array[bool] = []
```

**`begin_drag()` 新增两个末尾参数：**
```gdscript
func begin_drag(..., player_global_pos: Vector2,
        damage_labels: Array[String], damage_boosted: Array[bool]) -> void:
    ...
    _damage_labels = damage_labels.duplicate()
    _damage_boosted = damage_boosted.duplicate()
```

**`_cleanup()` 新增重置：**
```gdscript
_damage_labels.clear()
_damage_boosted.clear()
```

**`_draw()` 新增文字渲染：**

```gdscript
func _draw() -> void:
    if _state != State.DRAGGING or _ghost_card == null:
        return
    var from := _ghost_card.position + Vector2(GHOST_SIZE.x / 2.0, 0.0)
    var color := LINE_COLOR_SELF if _target_type == TARGET_NONE else LINE_COLOR_ENEMY
    for i in _target_line_ends.size():
        var target_pos: Vector2 = _target_line_ends[i]
        draw_dashed_line(from, target_pos, color, LINE_WIDTH, LINE_DASH)
        # 伤害预览文字
        var slot := _current_slot if _target_type == TARGET_SINGLE else i
        if slot >= 0 and slot < _damage_labels.size() and _damage_labels[slot] != "":
            var label_text := _damage_labels[slot]
            var label_color := Color(0.3, 1.0, 0.4) if _damage_boosted[slot] else Color.WHITE
            var mid := (from + target_pos) * 0.5
            draw_string(ThemeDB.fallback_font, mid, label_text,
                HORIZONTAL_ALIGNMENT_CENTER, -1, 20, label_color)
```

## 颜色规范

| 场景 | 颜色 |
|------|------|
| 伤害数字（无加成） | `Color.WHITE` |
| 伤害数字（有加成：脆弱/剑意） | `Color(0.3, 1.0, 0.4)`（与 LINE_COLOR_SELF 一致） |
| 无伤害牌（target_none） | 不显示数字 |
