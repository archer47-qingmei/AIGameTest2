# v0.45.0 玩家受击动画 + 战斗界面布局重构 设计文档

**目标：** 重构战斗界面布局（右侧信息栏 + 敌人区扩展），并在玩家受到伤害时播放受击动画（红色闪烁 + 缩放抖动 + 伤害数字飞出）。

**架构：** `CombatEngine` 新增 `player_damaged(amount)` 信号；`CombatScreen` 监听信号，用 Tween 驱动 RightPanel 受击动画。布局改为 HBoxContainer 主结构，左侧主区域（敌人 + 手牌），右侧固定宽度信息栏（玩家状态 + 操作按钮）。

**Tech Stack：** Godot 4 GDScript，Tween API，严格类型。

### 新结构

```

│   ├── MainArea (VBoxContainer, SIZE_EXPAND_FILL 水平)
│   │   ├── EnemiesContainer (HBoxContainer, SIZE_EXPAND_FILL 垂直)
│   │   └── HandScroll (ScrollContainer, custom_minimum_size.y = 170)
│   └── RightPanel (VBoxContainer, custom_minimum_size.x = 160)
│       ├── LblPlayerHP (Label)
│       ├── LblPlayerBlock (Label)
│       ├── LblEnergy (Label)
│       ├── LblSwordIntent (Label)
│       ├── [LblRelics — 动态 add_child]
│       ├── Spacer (Control, SIZE_EXPAND_FILL 垂直)
│       ├── BtnEndTurn (Button)
│       └── BtnViewDeck (Button)
├── LblResult (Label, 锚点居中)

```

### 关键布局属性

- `HBoxContainer`：`anchors_preset = 15`（全屏），`size_flags` 默认
- `MainArea`（VBoxContainer）：`size_flags_horizontal = SIZE_EXPAND_FILL`
- `EnemiesContainer`（HBoxContainer）：`size_flags_vertical = SIZE_EXPAND_FILL`
- `HandScroll`：`custom_minimum_size = Vector2(0, 170)`
- `RightPanel`（VBoxContainer）：`custom_minimum_size = Vector2(160, 0)`，`size_flags_horizontal = SIZE_SHRINK_END`
- `Spacer`（Control）：`size_flags_vertical = SIZE_EXPAND_FILL`

## 功能 2：player_damaged 信号

### 新增信号（CombatEngine.gd）

```gdscript
signal player_damaged(amount: int)
```

### emit 位置 1：_do_enemy_turn()

在 `EffectResolver.apply_damage(enemies[i], player, action.value)` 前后做 HP 快照，有减少则 emit：

```gdscript
var hp_before: int = player.hp
EffectResolver.apply_damage(enemies[i], player, action.value)
var dmg: int = hp_before - player.hp
if dmg > 0:
    player_damaged.emit(dmg)
```

### emit 位置 2：end_turn() 毒液扣血

在 `player.hp = max(0, player.hp - venom_count)` 后 emit：

```gdscript
if venom_count > 0:
    player.hp = max(0, player.hp - venom_count)
    player_damaged.emit(venom_count)
    state_changed.emit()
```

## 功能 3：受击动画（CombatScreen）

### 连接信号

在 `_ready()` 中：

```gdscript
_engine.player_damaged.connect(_on_player_damaged)
```

### _on_player_damaged(amount: int)

动画目标：`_right_panel`（RightPanel VBoxContainer）。

```gdscript
func _on_player_damaged(amount: int) -> void:
    _right_panel.pivot_offset = _right_panel.size / 2

    var flash_tween: Tween = create_tween()
    flash_tween.tween_property(_right_panel, "modulate", Color(1.0, 0.2, 0.2), 0.05)
    flash_tween.tween_property(_right_panel, "modulate", Color(1.0, 1.0, 1.0), 0.2)

    var shake_tween: Tween = create_tween()
    shake_tween.tween_property(_right_panel, "scale", Vector2(1.05, 0.95), 0.06)
    shake_tween.tween_property(_right_panel, "scale", Vector2(0.97, 1.03), 0.06)
    shake_tween.tween_property(_right_panel, "scale", Vector2(1.02, 0.99), 0.06)
    shake_tween.tween_property(_right_panel, "scale", Vector2(1.0, 1.0), 0.07)

    var lbl: Label = Label.new()
    lbl.text = "-%d" % amount
    lbl.add_theme_font_size_override("font_size", 24)
    lbl.pivot_offset = Vector2(15, 12)
    lbl.z_index = 10
    add_child(lbl)
    lbl.global_position = _right_panel.get_global_rect().get_center() - Vector2(15, 12)

    var direction: float = 1.0 if randi() % 2 == 0 else -1.0
    var drift_x: float = direction * randf_range(20.0, 60.0)

    var flyout_tween: Tween = create_tween().set_parallel(true)
    flyout_tween.tween_property(lbl, "scale", Vector2(1.5, 1.5), 0.15).set_ease(Tween.EASE_OUT)
    flyout_tween.tween_property(lbl, "position:x", lbl.position.x + drift_x, 0.4)
    flyout_tween.tween_property(lbl, "modulate:a", 0.0, 0.3).set_delay(0.1)
    flyout_tween.finished.connect(lbl.queue_free)
```

伤害数字显示为负数形式（`"-X"`），与敌人受伤（正数）形成区分。
