# v0.21.0 遗物系统 设计文档

**日期：** 2026-04-30
**目标：** 实现遗物系统，精英战和 Boss 战胜利后自动获得遗物，遗物效果分为「战斗开始时」和「每回合触发」两类。

---

## 核心决策

| 问题 | 决策 |
|---|---|
| 遗物如何获得 | 精英战胜利获得「燃烧宝石」，Boss 胜利获得「生命之环」，自动添加无需玩家选择 |
| 精英战奖励 | 遗物 + 卡牌（正常奖励流程不变） |
| 效果类型 | COMBAT_START（战斗开始触发一次）+ TURN_START（每回合触发） |
| 架构方案 | RelicEngine 静态工具类，镜像 EffectResolver 模式 |
| 精英节点 | NodeConfig.Type 新增 ELITE，node_b2 改为精英节点 |

## 变更一：数据层

### 新增 RelicData.gd（Resource）

```gdscript
class_name RelicData
extends Resource

enum Trigger { COMBAT_START, TURN_START }

var display_name: String = ""
var description: String = ""
var trigger: Trigger = Trigger.COMBAT_START
var effect_type: String = ""   # "energy" / "heal_hp"
var value: int = 0
```

### 新增 data/relics/ 目录，2 个 .tres 文件

| 文件 | display_name | trigger | effect_type | value |
|---|---|---|---|---|
| `data/relics/burning_gem.tres` | 燃烧宝石 | COMBAT_START | energy | 1 |
| `data/relics/life_ring.tres` | 生命之环 | TURN_START | heal_hp | 1 |

**燃烧宝石描述：** 每次战斗开始时，额外获得 1 点能量。
**生命之环描述：** 每回合开始时，恢复 1 点生命。

### 新增 data/enemies/elite_guard.tres（精英敌人）

| 字段 | 值 |
|---|---|
| display_name | 精英卫士 |
| hp | 45 |
| actions | attack 14 → attack 14 → block 8（循环） |

### 修改 NodeConfig.gd

### 修改 data/nodes/node_b2.tres

- `type` 从 `COMBAT` 改为 `ELITE`

## 变更二：逻辑层

### 新增 RelicEngine.gd（静态工具）

```gdscript
class_name RelicEngine
extends RefCounted

static func apply_combat_start(relics: Array[RelicData], engine: CombatEngine) -> void:
    for relic: RelicData in relics:
        if relic.trigger == RelicData.Trigger.COMBAT_START:
            _apply(relic, engine)

static func apply_turn_start(relics: Array[RelicData], engine: CombatEngine) -> void:
    for relic: RelicData in relics:
        if relic.trigger == RelicData.Trigger.TURN_START:
            _apply(relic, engine)

static func _apply(relic: RelicData, engine: CombatEngine) -> void:
    match relic.effect_type:
        "energy":  engine.energy += relic.value
        "heal_hp": engine.player.hp = mini(engine.player.hp + relic.value, engine.player.max_hp)
```

### 修改 CombatEngine.gd

新增字段：

```gdscript
var _relics: Array[RelicData] = []
```

`setup()` 新增参数 `relics: Array[RelicData] = []`，存入 `_relics`：

```gdscript
func setup(initial_deck: Array[CardData], enemy_data: EnemyData, initial_hp: int, max_hp: int, relics: Array[RelicData] = []) -> void:
    _relics = relics
    # ... 其余逻辑不变 ...
    _start_player_turn()
```

`_start_player_turn()` 调用两个钩子：

```gdscript
func _start_player_turn() -> void:
    player.weak = max(0, player.weak - 1)
    turn_number += 1
    energy = 3
    player.block = 0
    _draw_hand()
    if turn_number == 1:
        RelicEngine.apply_combat_start(_relics, self)
    RelicEngine.apply_turn_start(_relics, self)
    state_changed.emit()
```

### 修改 GameManager.gd

新增 `is_elite_node()`、`_grant_relic()`，更新 `end_combat()`：

```gdscript
func is_elite_node() -> bool:
    return player_state.current_node.config.type == NodeConfig.Type.ELITE

func _grant_relic(relic_data: RelicData) -> void:
    player_state.relics.append(relic_data.duplicate())

func end_combat(final_hp: int) -> void:
    player_state.hp = final_hp
    if is_elite_node():
        _grant_relic(preload("res://data/relics/burning_gem.tres"))
    elif is_final_node():
        _grant_relic(preload("res://data/relics/life_ring.tres"))
    if is_final_node():
        go_to_win()
    else:
        go_to_reward()
```

### 修改 map/MapScreen.gd

`_get_node_label()` 新增 ELITE 分支：

```gdscript
func _get_node_label(nd: NodeData) -> String:
    if nd.connections.is_empty():
        return "Boss 战斗"
    if nd.config.type == NodeConfig.Type.REST:
        return "休息站"
    if nd.config.type == NodeConfig.Type.ELITE:
        return "精英战斗"
    return "战斗"
```

## 完整流程

```
正常战斗胜利 → go_to_reward()（选卡）→ go_to_map()
精英战斗胜利 → 自动获得「燃烧宝石」→ go_to_reward()（选卡）→ go_to_map()
Boss 胜利    → 自动获得「生命之环」→ go_to_win()
```
