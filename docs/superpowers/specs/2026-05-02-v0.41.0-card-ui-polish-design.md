## 功能 1：单选卡牌逻辑

### 当前行为

`_on_card_pressed(index)` 中：若 `_pending_card_index >= 0`（已有选中），点击任意卡牌 → 取消选中。用户需再点一次才能选另一张。

### 新行为

点击任意卡牌 → 直接选中该卡，替换之前的选中状态。不再需要先取消再选择。

### 实现

在 `_on_card_pressed(index)` 中，删除"已有选中则取消"的早返回分支，始终执行选卡逻辑。

当前（伪代码）：
```gdscript
func _on_card_pressed(index: int) -> void:
    if _pending_card_index >= 0:
        _cancel_targeting()
        return
    # 选卡逻辑...
```

修改后：
```gdscript
func _on_card_pressed(index: int) -> void:
    if _pending_card_index >= 0:
        _cancel_targeting()
        # 不 return，继续执行选卡逻辑
    # 选卡逻辑...
```

## 功能 2：卡牌固定尺寸

### 卡牌 Button 尺寸

每张卡牌 Button 设置：
- `custom_minimum_size = Vector2(110, 150)`（比例约 0.73，接近标准扑克牌）
- `autowrap_mode = TextServer.AUTOWRAP_WORD_SMART`（文字自动换行）
- `vertical_alignment = VERTICAL_ALIGNMENT_TOP`（文字从顶部开始）

在 `_rebuild_hand()` 中创建 Button 时应用上述属性。

### 手牌区域高度

`HandScroll`（ScrollContainer）高度从 80px 改为 170px（卡牌 150px + 上下内边距 20px）。在 `CombatScreen.tscn` 中修改该节点的 `custom_minimum_size.y`。

### 文本格式

`CardData.get_description()` 已使用 `\n` 分隔卡牌名称和效果行，开启 autowrap 后自然换行，无需修改数据层。
