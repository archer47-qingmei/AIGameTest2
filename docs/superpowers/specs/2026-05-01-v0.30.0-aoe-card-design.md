# v0.30.0 群攻牌「横扫」实现设计

**目标：** 新增一张群体攻击牌「横扫」，对战场上所有存活敌人各造成 5 点伤害。

**架构：** 新增 `target_type = "all"` 分支，改动集中于 CombatEngine（循环出牌）和 CombatScreen（立即出牌，不进选目标模式）。EffectResolver 保持纯单目标解算器不变。

---

## 数据层

新建 `data/cards/whirlwind.tres`：

| 字段 | 值 |
|------|----|
| card_name | 横扫 |
| cost | 2 |
| target_type | "all" |
| effects | CardEffectData(type="damage", value=5) |

`GameManager.start_new_run()` 起始牌组加入 1 张横扫（与 bash / slash 同等处理）。

---

## 逻辑层（CombatEngine）

`play_card(card_index, target_index)` 在扣除能量、取得 card 之后，按 `target_type` 分路：

- `"all"`：对 `enemies` 中所有 `hp > 0` 的 Combatant 依次调用 `EffectResolver.resolve(card, player, enemy)`；弱化 / 易伤修正在每个敌人上独立计算。
- `"single"` / 其他：现有逻辑不变（`target = enemies[target_index]`，调用一次 resolve）。

draw / energy 引擎效果（`_apply_engine_effects`）在循环结束后只执行一次。其余流程（移出手牌、入弃牌堆、`state_changed`、`_check_end()`）不变。

---

## UI 层（CombatScreen）

`_on_card_pressed()` 新增分支：

```
target_type == "none"   → 立即出牌（现有）
target_type == "all"    → 立即出牌（新增，与 "none" 处理相同）
target_type == "single" → 进入选目标模式（现有）
```

`_set_targeting_mode()`、`_refresh_ui()`、敌人按钮逻辑均不需要改动。
