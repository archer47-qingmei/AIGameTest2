# v0.47.0 手牌扇形布局 Design

**Goal:** 将手牌区从 ScrollContainer 改为自定义平铺展开布局，所有牌始终可见、不依赖滚动，适配手机触屏操作。

**Architecture:** 替换 tscn 中的 ScrollContainer + HBoxContainer 为单个普通 Control 节点（HandArea），在 CombatScreen.gd 中用绝对坐标定位每张牌，动态计算步长使牌数多时叠压、牌数少时均匀分布。

**Tech Stack:** Godot 4 GDScript，严格类型，Control 绝对定位，resized 信号处理尺寸变化。

## 布局算法

```
常量：CARD_W = 110，CARD_H = 150，CONTAINER_H = 170，MIN_GAP = 8

W = HandArea.size.x
  （若 W == 0，回退到 get_viewport_rect().size.x）

n = 手牌数量

若 n == 0：直接返回
若 n == 1：step = 0

否则：
  natural_step = CARD_W + MIN_GAP          # 118，牌不叠压时的步长
  max_step     = (W - CARD_W) / (n - 1)   # 牌刚好铺满时的步长
  step         = min(natural_step, max_step)

total_w  = CARD_W + (n - 1) * step
start_x  = (W - total_w) / 2              # 整组水平居中
start_y  = (CONTAINER_H - CARD_H) / 2    # = 10，垂直居中

第 i 张牌：position = Vector2(start_x + i * step, start_y)
```

**叠压顺序：** 后加入树的按钮（索引大）绘制时在前，靠右的牌自然压在靠左的牌上，无需手动设置 z_index。

**尺寸变化：** 连接 `HandArea.resized` 到 `_layout_hand()`，只重新定位不重建按钮，处理屏幕旋转等场景。

## 代码结构

### _rebuild_hand()（修改）

```gdscript
func _rebuild_hand() -> void:
    for btn: Button in _hand_buttons:
        btn.queue_free()
    _hand_buttons.clear()
    for i in _engine.hand.size():
        var card: CardData = _engine.hand[i]
        var btn := Button.new()
        btn.custom_minimum_size = Vector2(110, 150)
        btn.size = Vector2(110, 150)
        btn.autowrap_mode = TextServer.AUTOWRAP_WORD_SMART
        btn.vertical_alignment = VERTICAL_ALIGNMENT_TOP
        btn.text = card.get_description()
        btn.pressed.connect(_on_card_pressed.bind(i))
        _hand_area.add_child(btn)
        _hand_buttons.append(btn)
    _layout_hand()
    if _pending_card_index >= 0:
        _set_targeting_mode(true)
```

### _layout_hand()（新增）

```gdscript
func _layout_hand() -> void:
    var n := _hand_buttons.size()
    if n == 0:
        return
    const CARD_W := 110.0
    const CARD_H := 150.0
    const CONTAINER_H := 170.0
    const MIN_GAP := 8.0
    var W := _hand_area.size.x
    if W == 0:
        W = get_viewport_rect().size.x
    var step := 0.0
    if n > 1:
        var natural_step := CARD_W + MIN_GAP
        var max_step := (W - CARD_W) / (n - 1)
        step = min(natural_step, max_step)
    var total_w := CARD_W + (n - 1) * step
    var start_x := (W - total_w) / 2.0
    var start_y := (CONTAINER_H - CARD_H) / 2.0
    for i in n:
        _hand_buttons[i].position = Vector2(start_x + i * step, start_y)
```

### @onready 和 _ready()（修改）

```gdscript
# 旧
@onready var _hand_container: HBoxContainer = $VBoxContainer/HandScroll/HandContainer

# 新
@onready var _hand_area: Control = $VBoxContainer/HandArea
```

在 `_ready()` 中连接 resized 信号：
```gdscript
_hand_area.resized.connect(_layout_hand)
```
