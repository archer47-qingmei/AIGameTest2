# v0.40.0 四个新遗物 设计文档

**目标：** 新增 4 个遗物（窃天、融道、天魔解体、回光返照），扩展 RelicData/RelicEngine 支持新效果类型。

**架构：** 在现有双效果字段体系上新增 4 个 EffectType 枚举值和 1 个 RelicData 标志字段；RelicEngine 新增对应处理分支；ShopScreen 新增卡牌购买限制检查；所有遗物效果通过 `.tres` 数据文件驱动。

**Tech Stack：** Godot 4 GDScript，Resource `.tres`，现有 RelicEngine / CombatEngine / GameManager 体系。

## 遗物设计详细

### 窃天
- **效果：** 真气上限 +1（每场战斗），不能从坊市购买卡牌
- **主效果：** trigger=0(COMBAT_START), effect_type=6(ENERGY_CAP), value=1
- **标志：** `blocks_card_purchase = true`
- **描述文本：** "真气上限 +1。不能从坊市购买卡牌。"

### 融道
- **效果：** 真气上限 +1（每场战斗），装备时随机移除牌组 2 张牌
- **主效果：** trigger=0(COMBAT_START), effect_type=6(ENERGY_CAP), value=1
- **副效果：** trigger_b=4(ON_EQUIP), effect_type_b=10(REMOVE_CARDS), value_b=2
- **描述文本：** "真气上限 +1。装备时随机移除牌组中 2 张牌。"

### 天魔解体
- **效果：** 第 1 回合额外获得 5 点真气；第 2 回合真气 -1（最低 2）
- **主效果：** trigger=1(TURN_START), effect_type=11(FIRST_TURN_ENERGY), value=5
- **副效果：** trigger_b=1(TURN_START), effect_type_b=12(SECOND_TURN_ENERGY_PENALTY), value_b=1
- **描述文本：** "第 1 回合获得 5 点额外真气。第 2 回合真气 -1（最低 2）。"

### 回光返照
- **效果：** 每回合开始，若 HP < 最大血量 30%，获得 3 点真气；最大血量 -20%（装备时立即生效）
- **主效果：** trigger=1(TURN_START), effect_type=13(ENERGY_IF_LOW_HP), value=3
- **副效果：** trigger_b=4(ON_EQUIP), effect_type_b=7(MAX_HP_PERCENT), value_b=20
- **描述文本：** "每回合开始若 HP 低于 30%，获得 3 点真气。最大血量 -20%（立即生效）。"

## RelicEngine 新增逻辑

### `_apply_effect` 新增分支

```gdscript
RelicData.EffectType.FIRST_TURN_ENERGY:
    if engine.turn_number == 1:
        engine.energy += value

RelicData.EffectType.SECOND_TURN_ENERGY_PENALTY:
    if engine.turn_number == 2:
        engine.energy = max(2, engine.energy - value)

RelicData.EffectType.ENERGY_IF_LOW_HP:
    if engine.player.hp < int(engine.player.max_hp * 0.3):
        engine.energy += value
```

### `_apply_equip_effect` 新增分支

```gdscript
RelicData.EffectType.REMOVE_CARDS:
    var deck: Array[CardData] = player_state.deck
    for i in value:
        if deck.is_empty():
            break
        deck.remove_at(randi() % deck.size())
```

### ShopScreen `_rebuild_card_list` 新增检查

```gdscript
var card_purchase_blocked: bool = GameManager.player_state.relics.any(
    func(r: RelicData) -> bool: return r.blocks_card_purchase
)
# 卡牌行的 btn.disabled 条件加入 or card_purchase_blocked
```
