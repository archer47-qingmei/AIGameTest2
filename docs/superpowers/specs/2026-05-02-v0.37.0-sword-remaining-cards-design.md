# v0.37.0 剑修剩余卡牌设计

**目标：** 新增 6 张剑修卡牌（凡品 3 张 / 灵品 3 张），扩展 effect 系统支持回合内状态追踪、下回合延迟触发、被动响应触发和条件性剑意保留，将全部 21 张剑修牌纳入奖励池。

---

## 卡牌一览

| 牌名 | 品级 | 类型 | 费用 | 效果 |
|------|------|------|------|------|
| 剑鞘 | 凡品 | 身法 | 1 | 格挡7；若本回合未打出过招式，获得1层剑意 |
| 剑气盈体 | 凡品 | 功法 | 1 | 下回合开始获得2层剑意 |
| 惊鸿 | 凡品 | 招式·终结技 | 2 | 打7，消耗全部剑意；若目标意图为攻击，不消耗剑意 |
| 剑步 | 灵品 | 身法 | 1 | 格挡6；下回合开始额外抽1张牌 |
| 人剑合一 | 灵品 | 功法 | 2 | 本场战斗：每次打出终结技获得3点格挡 |
| 意随心发 | 灵品 | 功法 | 1 | 本场战斗：每回合第一次获得剑意时额外获得2点格挡 |

## 数据层

### Combatant.gd

在现有字段之后新增 7 个运行时字段：

```gdscript
var played_style_this_turn: bool = false       # 剑鞘：本回合是否打过招式
var gained_sword_intent_this_turn: bool = false # 意随心发：本回合是否已获得过剑意
var first_si_block_bonus: int = 0              # 意随心发：首次获剑意时额外格挡
var next_turn_sword_intent: int = 0            # 剑气盈体：下回合开始增加的剑意
var next_turn_draw: int = 0                    # 剑步：下回合开始额外抽牌数
var finisher_block_bonus: int = 0              # 人剑合一：每次打出终结技获得的格挡
var current_intent: String = ""               # 惊鸿：敌人当前意图（仅 enemy 使用）
```

所有字段随 `Combatant.new()` 自动初始化，每场战斗自动重置，无需额外逻辑。

### CardData.gd

新增一个字段：

```gdscript
@export var retain_si_if_target_attacks: bool = false
```

用于惊鸿：终结技消耗剑意前检查目标意图，若为攻击则跳过消耗。

## 逻辑层

### EffectResolver.gd

**改动 1 — 招式获得剑意路径（末尾，替换原有 elif）：**

```gdscript
elif card.card_type == "招式":
    var will_gain: bool = attacker.sword_intent < attacker.sword_intent_cap
    if will_gain and not attacker.gained_sword_intent_this_turn:
        attacker.add_block(attacker.first_si_block_bonus)
        attacker.gained_sword_intent_this_turn = true
    attacker.sword_intent = mini(attacker.sword_intent + 1, attacker.sword_intent_cap)
    attacker.played_style_this_turn = true
```

**改动 2 — 终结技路径（替换原有 if card.is_finisher）：**

```gdscript
if card.is_finisher:
    attacker.add_block(attacker.finisher_block_bonus)
    var skip_consume: bool = card.retain_si_if_target_attacks \
        and defender != null and defender.current_intent == "attack"
    if not skip_consume:
        attacker.sword_intent = 0
```

### CombatEngine.gd

**改动 1 — `_start_player_turn()` 开头插入：**

```gdscript
player.played_style_this_turn = false
player.gained_sword_intent_this_turn = false
player.sword_intent = mini(
    player.sword_intent + player.next_turn_sword_intent,
    player.sword_intent_cap
)
player.next_turn_sword_intent = 0
```

在 `_draw_hand()` 之后、现有 `draw_per_turn` 逻辑之后插入：

```gdscript
if player.next_turn_draw > 0:
    _draw_cards(player.next_turn_draw)
    player.next_turn_draw = 0
```

在 `RelicEngine` 调用之前插入（设置敌人意图）：

```gdscript
for i in enemies.size():
    if enemies[i].hp > 0:
        enemies[i].current_intent = get_enemy_action(i).type
```

**改动 2 — `_apply_engine_effects()` 新增 5 个分支：**

```gdscript
elif effect.type == "sword_intent_if_no_style":
    if not player.played_style_this_turn:
        player.sword_intent = mini(player.sword_intent + effect.value, player.sword_intent_cap)
elif effect.type == "next_turn_si":
    player.next_turn_sword_intent += effect.value
elif effect.type == "next_turn_draw":
    player.next_turn_draw += effect.value
elif effect.type == "finisher_block":
    player.finisher_block_bonus += effect.value
elif effect.type == "first_si_block":
    player.first_si_block_bonus += effect.value
```

## UI 层

### CardData.get_description() 更新

扫描 effect 新增：

```gdscript
var next_turn_si: int = 0
var next_turn_drw: int = 0
var finisher_blk: int = 0
var first_si_blk: int = 0
var si_if_no_style: int = 0

elif effect.type == "next_turn_si":              next_turn_si = effect.value
elif effect.type == "next_turn_draw":            next_turn_drw = effect.value
elif effect.type == "finisher_block":            finisher_blk = effect.value
elif effect.type == "first_si_block":            first_si_blk = effect.value
elif effect.type == "sword_intent_if_no_style":  si_if_no_style = effect.value
```

描述行新增：

```gdscript
if next_turn_si > 0:    desc += "  下回合剑意+%d" % next_turn_si
if next_turn_drw > 0:   desc += "  下回合抽+%d" % next_turn_drw
if finisher_blk > 0:    desc += "  终结技触发挡+%d" % finisher_blk
if first_si_blk > 0:    desc += "  首剑意+挡%d" % first_si_blk
if si_if_no_style > 0:  desc += "  无招式时+剑意%d" % si_if_no_style
if retain_si_if_target_attacks: desc += "  攻击意图→剑意不消耗"
```
