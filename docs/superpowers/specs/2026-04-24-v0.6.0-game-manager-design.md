# v0.6.0 GameManager 与主菜单设计文档

**日期**：2026-04-24  
**项目**：AIGameTest2（Godot 4.6 + GDScript）  
**范围**：v0.6.0 — 引入 GameManager Autoload，新增主菜单场景，建立完整游戏流程（主菜单 → 战斗 → 主菜单）

---

## 背景

v0.5.0 完成后，游戏仍直接启动战斗场景，没有入口和出口。v0.6.0 目标：建立最小可运行的完整游戏循环框架，为后续地图和奖励系统打基础。

---

## 架构原则遵守情况

- **GameManager** 作为 Autoload Node——场景切换需要 SceneTree，这是合理的基础设施层例外，不含可测试的游戏规则逻辑
- **MainMenu.gd** 纯 UI 层，只转发按钮事件给 GameManager
- **CombatScreen.gd** 不直接知道主菜单路径，通过 `GameManager.go_to_menu()` 解耦

---

## 文件结构

```

```

## 模块设计

### GameManager.gd（Autoload）

```gdscript
class_name GameManager
extends Node

enum Phase { MENU, COMBAT }

var current_phase: Phase = Phase.MENU

func go_to_combat() -> void:
    current_phase = Phase.COMBAT
    get_tree().change_scene_to_file("res://combat/CombatScreen.tscn")

func go_to_menu() -> void:
    current_phase = Phase.MENU
    get_tree().change_scene_to_file("res://menu/MainMenu.tscn")
```

**接口：**
- `go_to_combat()` — 任何场景调用，切换到战斗
- `go_to_menu()` — 任何场景调用，切换到主菜单
- `current_phase` — 当前阶段只读状态

### menu/MainMenu.gd

```gdscript
extends Control

@onready var _btn_start: Button = $BtnStart

func _ready() -> void:
    _btn_start.pressed.connect(_on_start_pressed)

func _on_start_pressed() -> void:
    GameManager.go_to_combat()
```

### combat/CombatScreen.gd（改动部分）

新增：
```gdscript
@onready var _btn_return_menu: Button = $BtnReturnMenu

func _ready() -> void:
    # ...（现有代码不变）
    _btn_return_menu.pressed.connect(GameManager.go_to_menu)
```

`_on_combat_ended` 改为：
```gdscript
func _on_combat_ended(result: String) -> void:
    _lbl_result.text = "胜利！" if result == "victory" else "游戏结束"
    _lbl_result.visible = true
    _btn_end_turn.disabled = true
    _btn_return_menu.visible = true
    for btn: Button in _hand_buttons:
        btn.disabled = true
```

### project.godot 改动

```
run/main_scene="res://menu/MainMenu.tscn"

[autoload]
GameManager="*res://GameManager.gd"
```

## 游戏流程

```
启动
  └─→ MainMenu.tscn
        └─→ 点击"开始游戏" → GameManager.go_to_combat()
              └─→ CombatScreen.tscn
                    ├─→ 战斗进行中...
                    └─→ 战斗结束（胜利/失败）
                          └─→ 点击"返回主菜单" → GameManager.go_to_menu()
                                └─→ MainMenu.tscn
```
