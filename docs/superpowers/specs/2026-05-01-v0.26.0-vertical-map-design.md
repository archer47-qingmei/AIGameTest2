# 竖向地图 Design Spec

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 将地图界面改为竖向布局，起点在屏幕底部，Boss 在顶部，适配已有的 480×854 竖屏分辨率。

**Architecture:** 只修改 `map/MapScreen.gd` 的 `_get_node_pos()` 函数，将列号（column）映射到 Y 轴、行号（row）映射到 X 轴。其余逻辑（按钮生成、连线绘制、标签显示）无需修改。

**Tech Stack:** Godot 4 GDScript，项目分辨率已为 480×854。

---

## 坐标映射

`_get_node_pos()` 新逻辑：

| column | row | 新坐标 |
|--------|-----|--------|
| 3 (Boss) | — | (240, 150) |
| 2 | 0 | (140, 320) |
| 2 | 1 | (340, 320) |
| 1 | 0 | (140, 490) |
| 1 | 1 | (340, 490) |
| 0 | 0 | (140, 660) |
| 0 | 1 | (340, 660) |

新函数实现：

```gdscript
func _get_node_pos(nd: NodeData) -> Vector2:
	if nd.config.column == 3:
		return Vector2(240.0, 150.0)
	var row_x: float = 140.0 if nd.config.row == 0 else 340.0
	var col_y: float
	match nd.config.column:
		0: col_y = 660.0
		1: col_y = 490.0
		2: col_y = 320.0
		_: col_y = 660.0
	return Vector2(row_x, col_y)
```
