## 数据层

### CardEffectData.gd

新增字段：

```gdscript
@export var times: int = 1
```

表示该 effect 触发次数。默认 1，向下兼容。碎星的伤害 effect 设 `times=2`。

### CardData.gd

新增字段：

```gdscript
@export var draw_after_finisher: int = 0
```

终结技消耗剑意后额外抽牌数。破军设为 1，其余终结技保持 0。

## 逻辑层

### Combatant.gd

在现有剑意字段之后新增三个运行时字段：

```gdscript
var draw_per_turn: int = 0
var sword_intent_retain: bool = false
var sword_intent_block_bonus: int = 0
```

`Combatant.new()` 每场战斗重建，三个字段自动从默认值开始，无需额外重置逻辑。

### EffectResolver.gd

`resolve(card, attacker, defender, engine)` 签名新增可选参数 `engine: CombatEngine = null`，仅抽牌场景使用。

**改动 1 — times 循环：**

每条 effect 的执行外包一层循环：

```gdscript
for _i in effect.times:
    # 原有 if/elif 分支
```

**改动 2 — draw_card effect type：**

```gdscript
elif effect.type == "draw_card":
    if engine != null:
        engine.draw_cards(effect.value)
```

**改动 3 — 身法格挡剑意加成：**

block effect 结算时：

```gdscript
elif effect.type == "block":
    var bonus: int = attacker.sword_intent * attacker.sword_intent_block_bonus
    attacker.add_block(effect.value + bonus)
```

**改动 4 — draw_after_finisher：**

finisher 消耗剑意后（`attacker.sword_intent = 0` 之后）：

```gdscript
if card.draw_after_finisher > 0 and engine != null:
    engine.draw_cards(card.draw_after_finisher)
```

### CombatEngine.gd

**改动 1 — end_turn() 剑意折半条件：**

```gdscript
if not player.sword_intent_retain:
    player.sword_intent = player.sword_intent / 2
```

**改动 2 — start_turn() 额外抽牌：**

现有抽牌逻辑执行完后追加：

```gdscript
if player.draw_per_turn > 0:
    draw_cards(player.draw_per_turn)
```

**改动 3 — _apply_engine_effects() 新增三个分支：**

```gdscript
elif effect.type == "draw_per_turn":
    player.draw_per_turn += effect.value
elif effect.type == "sword_intent_retain":
    player.sword_intent_retain = true
elif effect.type == "sword_intent_block_bonus":
    player.sword_intent_block_bonus += effect.value
```

## 不在本版范围内

- 惊鸿（需要读取敌人意图判断）
- 剑鞘（需要记录回合内是否打出过招式）
- 剑气盈体（需要"下回合开始"延迟触发）
- 磨剑（需要运行时修改手牌费用）
- 剑心通明、回风（需要"本回合临时翻倍"状态）
- 所有灵品身法（剑步、回风）
- 淬剑（需要扫描牌组动态改费）
- 一剑（X 费牌）
- 杀意（消耗型功法 + 负面效果）
