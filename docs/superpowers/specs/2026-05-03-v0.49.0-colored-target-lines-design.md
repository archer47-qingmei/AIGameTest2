# v0.49.0 彩色拖拽连线 Design

**Goal:** 将拖拽出牌时的白色虚线改为彩色：攻击类牌（single/all）连线敌人显示红色，自身牌（none）连线玩家面板显示绿色。

**Architecture:** 在现有 DragLayer 基础上最小改动——`begin_drag()` 新增 `player_global_pos` 参数，`_draw()` 按 `_target_type` 决定颜色和目标点。CombatScreen 调用时传入玩家面板中心坐标。

**Tech Stack:** Godot 4 GDScript，严格类型，CanvasItem._draw() 虚线绘制。

---

## 改动范围

| 文件 | 动作 | 变更内容 |
|------|------|----------|
| `combat/DragLayer.gd` | 修改 | 新增 `LINE_COLOR_ENEMY`、`LINE_COLOR_SELF` 常量；`begin_drag()` 加 `player_global_pos` 参数；新增 `_player_local_pos` 字段；`_draw()` 按 target_type 选颜色和目标；`_cleanup()` 重置 `_player_local_pos` |
| `combat/CombatScreen.gd` | 修改 | `_on_card_button_down()` 中调用 `begin_drag()` 时额外传 `_player_card.get_global_rect().get_center()` |
| `menu/MainMenu.tscn` | 修改 | 版本号 v0.48.0 → v0.49.0 |

CombatEngine、CardData、EffectResolver、DragLayer 状态机逻辑不动。

## 连线颜色规则

| target_type | 连线颜色 | 连线目标 |
|---|---|---|
| `single` | 红色 `Color(1.0, 0.3, 0.3, 0.85)` | 当前锁定敌人（`_target_line_ends[0]`） |
| `all` | 红色 `Color(1.0, 0.3, 0.3, 0.85)` | 所有存活敌人（`_target_line_ends` 全部） |
| `none` | 绿色 `Color(0.3, 1.0, 0.4, 0.85)` | 玩家面板中心（`_player_local_pos`） |

## DragLayer.gd 接口变更

```gdscript
# 新增常量
const LINE_COLOR_ENEMY := Color(1.0, 0.3, 0.3, 0.85)
const LINE_COLOR_SELF  := Color(0.3, 1.0, 0.4, 0.85)

# 新增字段
var _player_local_pos: Vector2 = Vector2.ZERO

# begin_drag 新增最后一个参数
func begin_drag(card_index: int, card_global_pos: Vector2, card_text: String,
        target_type: String, enemy_global_positions: Array[Vector2],
        enemy_engine_indices: Array[int], player_global_pos: Vector2) -> void:
    # ... 原有逻辑不变 ...
    _player_local_pos = get_global_transform().affine_inverse() * player_global_pos
    if target_type == TARGET_NONE:
        _target_line_ends = [_player_local_pos]
```

---

## _draw() 变更

```gdscript
func _draw() -> void:
    if _state != State.DRAGGING or _ghost_card == null:
        return
    var from := _ghost_card.position + Vector2(GHOST_SIZE.x / 2.0, 0.0)
    var color := LINE_COLOR_ENEMY if _target_type != TARGET_NONE else LINE_COLOR_SELF
    for target_pos: Vector2 in _target_line_ends:
        draw_dashed_line(from, target_pos, color, LINE_WIDTH, LINE_DASH)
```
