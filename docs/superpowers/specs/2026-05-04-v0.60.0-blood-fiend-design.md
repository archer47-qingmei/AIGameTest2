# 血煞 实现设计 (v0.60.0)

## 目标

新增第三只精英怪：血煞，引入吸血攻击、噬子（击杀友方回血）、召唤（复活小血煞）三种新机制。

---

## 怪物数据

### 血煞

| 属性 | 值 |
|------|-----|
| HP | 128 |
| 行动模式 | 随机权重 |

**行动表：**

| 动作 | 类型 | 权重 | 参数 |
|------|------|------|------|
| 吸血攻击 | `vampiric_attack` | 45 | value=10, count=5 |
| 噬子 | `devour_minion` | 25 | — |
| 召唤 | `revive_minions` | 20 | — |
| 防御 | `block` | 10 | value=8 |

**`vampiric_attack`：** 对玩家造成 value=10 伤害，自回 count=5 HP（上限 max_hp）。回血量为固定值，不受格挡影响。

**`devour_minion`：** 找当前存活友方（enemies 数组中 index≠self，hp>0）中血量最低的一只，将其 HP 置为 0（视为死亡），自回 20 HP（上限 max_hp），自身获得 3 力量。若无存活友方，此动作无效（浪费回合）。

**`revive_minions`：** 将 enemies 数组中所有 hp≤0 的友方（index≠self）恢复至各自的 max_hp。若所有友方均存活，此动作无效（浪费回合）。

### 小血煞

| 属性 | 值 |
|------|-----|
| HP | 35 |
| 行动模式 | 顺序循环 |

**意图循环（1 个动作，永远循环）：**

| 回合 | 动作 | 效果 |
|------|------|------|
| 每回合 | `attack` | 打 5 |

### 出场与死亡规则

- 战斗开始：血煞 + 小血煞×2 全部出场（EnemyGroupData 包含三只）
- 血煞死亡后，残余小血煞继续战斗，玩家需全部击杀才算胜利
- 小血煞被噬子吃掉后视为死亡（hp=0），可被召唤复活至满血（35 HP）

## 架构改动

### 新增 action 类型（CombatEngine._do_enemy_turn）

**`vampiric_attack`：**
```gdscript
"vampiric_attack":
    var hp_before: int = player.hp
    EffectResolver.apply_damage(enemies[i], player, action.value)
    var dmg: int = hp_before - player.hp
    if dmg > 0:
        player_damaged.emit(dmg)
    enemies[i].hp = mini(enemies[i].hp + action.count, enemies[i].max_hp)
    enemies[i].weak = max(0, enemies[i].weak - 1)
```

**`devour_minion`：**
```gdscript
"devour_minion":
    var target_idx: int = -1
    var lowest_hp: int = 9999
    for j in enemies.size():
        if j != i and enemies[j].hp > 0 and enemies[j].hp < lowest_hp:
            lowest_hp = enemies[j].hp
            target_idx = j
    if target_idx >= 0:
        enemies[target_idx].hp = 0
        enemies[i].hp = mini(enemies[i].hp + 20, enemies[i].max_hp)
        enemies[i].strength += 3
```

同血量时取 index 最小的那只（遍历从低到高，`<` 而非 `<=`，自然保留最早遇到的）。

**`revive_minions`：**
```gdscript
"revive_minions":
    for j in enemies.size():
        if j != i and enemies[j].hp <= 0:
            enemies[j].hp = enemies[j].max_hp
            enemies[j].block = 0
```

### 蓄力翻倍（_resolve_enemy_actions）

`vampiric_attack` 加入蓄力检测列表（与 `attack` 等同处理）：
```gdscript
if enemies[i].is_charging and action.type in ["attack", "attack_weak", "attack_vulnerable", "multi_attack", "attack_curse", "vampiric_attack"]:
```

### intent 文字（CombatScreen._intent_text）

```gdscript
"vampiric_attack":
    return "意图：吸血 %d" % val
"devour_minion":
    return "意图：噬子"
"revive_minions":
    return "意图：召唤"
```
