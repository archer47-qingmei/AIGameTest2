# v0.11.0 多敌人地图 设计文档

**日期：** 2026-04-28
**目标：** 地图扩展为 3 节点，普通敌人随机排列，新增蓄力爆发型敌人，增强每局重玩差异性。

---

## 核心决策

| 问题 | 决策 |
|---|---|
| 节点数 | 3 个（2 普通 + 1 Boss 固定压轴） |
| 新敌人数量 | 1 个（烈焰蜥蜴） |
| 序列生成时机 | `start_new_run()` 时生成，存入 `PlayerState` |
| 地图显示 | 只显示"关卡 X"，不透露敌人信息 |

---

## 架构总览

```
数据层
  PlayerState.gd         新增 enemy_sequence: Array[String]
  data/enemies/fire_lizard.tres  新增蓄力爆发型敌人

逻辑层
  GameManager.gd         新增 NORMAL_POOL / BOSS_ENEMY 常量；
                         start_new_run() 生成并写入序列；
                         get_current_enemy_data() 改读序列；
                         删除 STAGE_ENEMIES 常量

UI 层
  MapScreen.gd           节点数动态读 enemy_sequence.size()；
                         标签改为"关卡 X" / "关卡 X（Boss）"；
                         删除硬编码 NODE_LABELS
```

**数据流：**

```
start_new_run()
  → shuffle(NORMAL_POOL) 取前 2 + append(BOSS_ENEMY)
  → PlayerState.enemy_sequence = [..., "res://data/enemies/boss.tres"]

go_to_combat()
  → get_current_enemy_data()
  → load(enemy_sequence[current_node])
```

### data/enemies/fire_lizard.tres（新增）

- display_name: `"烈焰蜥蜴"`
- hp: `38`
- 行动循环（2 回合）：
  - 第 1 回合：block 8（蓄力）
  - 第 2 回合：attack 18（爆发）

设计意图：与颚虫（攻 11 / 挡 6 交替）形成策略互补——颚虫考验稳定格挡，烈焰蜥蜴考验节奏预判。

## 逻辑层（GameManager.gd）

**删除：** `STAGE_ENEMIES` 常量

**新增常量：**
```gdscript
const NORMAL_POOL: Array[String] = [
    "res://data/enemies/jaw_worm.tres",
    "res://data/enemies/fire_lizard.tres",
]
const BOSS_ENEMY: String = "res://data/enemies/boss.tres"
```

**`start_new_run()` 新增序列生成：**
```gdscript
var pool := NORMAL_POOL.duplicate()
pool.shuffle()
player_state.enemy_sequence = pool.slice(0, 2)
player_state.enemy_sequence.append(BOSS_ENEMY)
```

**`get_current_enemy_data()` 改为：**
```gdscript
func get_current_enemy_data() -> EnemyData:
    return load(player_state.enemy_sequence[player_state.current_node]) as EnemyData
```

**`is_final_node()` 改为：**
```gdscript
func is_final_node() -> bool:
    return player_state.current_node == player_state.enemy_sequence.size() - 1
```

## UI 层（MapScreen.gd）

**删除：** `NODE_LABELS` 常量

**`_ready()` 改为动态生成：**
```gdscript
func _ready() -> void:
    var current: int = GameManager.player_state.current_node
    var total: int = GameManager.player_state.enemy_sequence.size()
    for i: int in total:
        var btn: Button = Button.new()
        var label: String = "关卡 %d" % (i + 1)
        if i == total - 1:
            label = "关卡 %d（Boss）" % (i + 1)
        if i < current:
            btn.text = label + "（已完成）"
            btn.disabled = true
        elif i == current:
            btn.text = label
            btn.pressed.connect(GameManager.go_to_combat)
        else:
            btn.text = label + "（未解锁）"
            btn.disabled = true
        _node_container.add_child(btn)
```
