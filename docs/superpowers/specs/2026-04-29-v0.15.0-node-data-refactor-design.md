# v0.15.0 NodeData 重构 设计文档

**日期：** 2026-04-29
**目标：** 还清 TD-002——将 `enemy_sequence: Array[String]` 升级为 `node_sequence: Array[NodeData]`，消除魔法字符串，使节点类型显式且类型安全。

---

## 核心决策

| 问题 | 决策 |
|---|---|
| 节点类型表达方式 | `NodeData.Type` enum（COMBAT / REST），不用字符串或 null 语义 |
| NodeData 实例化 | 运行时 `NodeData.new()` 构造，不需要新 `.tres` 文件 |
| EnemyData .tres 文件 | 保持不变，NodeData 持有对其的直接引用 |
| `enemy_sequence` 改名 | 改为 `node_sequence`，语义准确 |

---

## 架构总览

```
数据层（新增）
  NodeData.gd       Type enum（COMBAT/REST）+ enemy_data: EnemyData

逻辑层（修改）
  PlayerState.gd    enemy_sequence: Array[String] → node_sequence: Array[NodeData]
  GameManager.gd    构造 NodeData 对象；移除 REST_NODE 常量；更新相关方法

UI 层（修改）
  map/MapScreen.gd  sequence 类型改为 Array[NodeData]；类型判断改为 node.type enum
```

**数据流（修改后）：**

```
start_new_run()
  → 构造 NodeData(COMBAT, jaw_worm 或 fire_lizard)（随机 2 选 1）
  → 构造 NodeData(REST)
  → 构造 NodeData(COMBAT, boss)
  → player_state.node_sequence = [combat, combat, rest, combat(boss)]

MapScreen 显示节点
  → sequence[i].type == NodeData.Type.REST → 标签"休息站"，路由 go_to_rest
  → 否则 → 标签"关卡 N"或"关卡 N（Boss）"，路由 go_to_combat

get_current_enemy_data()
  → player_state.node_sequence[current_node].enemy_data
```

## 各文件详细设计

### NodeData.gd（新增）

```gdscript
class_name NodeData
extends Resource

enum Type { COMBAT, REST }

var type: Type = Type.COMBAT
var enemy_data: EnemyData
```

REST 节点的 `enemy_data` 保持未赋值（null）。

### PlayerState.gd

只改一行：

```gdscript
var node_sequence: Array[NodeData] = []
```

### GameManager.gd

移除 `REST_NODE: String = "rest"` 常量。`NORMAL_POOL` 和 `BOSS_ENEMY` 字符串常量保留（仍用于 load EnemyData）。

`start_new_run()` 序列构造：

```gdscript
var pool: Array[NodeData] = []
for path: String in NORMAL_POOL:
    var nd: NodeData = NodeData.new()
    nd.type = NodeData.Type.COMBAT
    nd.enemy_data = load(path) as EnemyData
    pool.append(nd)
pool.shuffle()
player_state.node_sequence.assign(pool)

var rest_nd: NodeData = NodeData.new()
rest_nd.type = NodeData.Type.REST
player_state.node_sequence.append(rest_nd)

var boss_nd: NodeData = NodeData.new()
boss_nd.type = NodeData.Type.COMBAT
boss_nd.enemy_data = load(BOSS_ENEMY) as EnemyData
player_state.node_sequence.append(boss_nd)
```

`get_current_enemy_data()`：

```gdscript
func get_current_enemy_data() -> EnemyData:
    return player_state.node_sequence[player_state.current_node].enemy_data
```

`is_final_node()`：

```gdscript
func is_final_node() -> bool:
    return player_state.current_node == player_state.node_sequence.size() - 1
```

### map/MapScreen.gd

```gdscript
func _ready() -> void:
    var current: int = GameManager.player_state.current_node
    var sequence: Array[NodeData] = GameManager.player_state.node_sequence
    var total: int = sequence.size()
    for i: int in total:
        var btn: Button = Button.new()
        var label: String = _get_node_label(i, sequence, total)
        if i < current:
            btn.text = label + "（已完成）"
            btn.disabled = true
        elif i == current:
            btn.text = label
            if sequence[i].type == NodeData.Type.REST:
                btn.pressed.connect(GameManager.go_to_rest)
            else:
                btn.pressed.connect(GameManager.go_to_combat)
        else:
            btn.text = label + "（未解锁）"
            btn.disabled = true
        _node_container.add_child(btn)

func _get_node_label(i: int, sequence: Array[NodeData], total: int) -> String:
    if sequence[i].type == NodeData.Type.REST:
        return "休息站"
    elif i == total - 1:
        return "关卡 %d（Boss）" % (i + 1)
    else:
        return "关卡 %d" % (i + 1)
```
