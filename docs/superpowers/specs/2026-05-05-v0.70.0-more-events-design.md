# v0.70.0 机缘事件扩充 Implementation Design

**日期：** 2026-05-05

---

## Goal

在 v0.69.0 事件框架基础上，将 3 件已有事件升级为三选项版本，并新增 7 件事件，同时补全所需引擎支持。

## Architecture

三层不变（数据层 `.tres` / 逻辑层 `EventEngine` / UI 层 `EventScreen`）。本版本只扩充数据和逻辑层，不改 UI 层。

新增引擎能力：`LOSE_RELIC_RANDOM`、`GAIN_ENERGY_CAP`、`REMOVE_CARDS_RANDOM`，以及跨系统的真气上限持久化（`PlayerState.energy_cap` → `CombatEngine.setup()`）。

## 第一部分：引擎/基础层

### 1. 业火诅咒牌

`data/cards/ye_huo.tres`：`is_curse = true`，费用 0，`card_name = "业火"`，`special_text = "无法通过休息站移除。每次战斗开始时受到 2 点伤害。"` （效果触发留待后续版本实装，本版先有数据文件即可）

### 2. PlayerState.energy_cap

`PlayerState.gd` 新增字段：
```gdscript
var energy_cap: int = 3
```

### 3. CombatEngine.setup() 传入 energy_cap

`CombatEngine.gd`：
```gdscript
func setup(initial_deck: Array[CardData], enemy_group: EnemyGroupData, initial_hp: int, max_hp: int, relics: Array[RelicData] = [], initial_energy_cap: int = 3) -> void:
    energy_cap = initial_energy_cap
    # 其余不变
```

`CombatScreen.gd` `_engine.setup(...)` 末尾加参数：
```gdscript
_engine.setup(
    GameManager.player_state.deck,
    GameManager.get_current_enemy_group(),
    GameManager.player_state.hp,
    GameManager.player_state.max_hp,
    GameManager.player_state.relics,
    GameManager.player_state.energy_cap
)
```

### 4. EventEffectData.curse_card_path

```gdscript
@export var curse_card_path: String = ""
```

### 5. EventEngine 补充效果

`_apply_effect` 新增三个 case：

```gdscript
EventEffectData.EffectType.LOSE_RELIC_RANDOM:
    if not player_state.relics.is_empty():
        var idx: int = randi() % player_state.relics.size()
        player_state.relics.remove_at(idx)

EventEffectData.EffectType.GAIN_ENERGY_CAP:
    player_state.energy_cap += effect.value

EventEffectData.EffectType.REMOVE_CARDS_RANDOM:
    var count: int = mini(effect.value, player_state.deck.size())
    for _i in count:
        var idx: int = randi() % player_state.deck.size()
        player_state.deck.remove_at(idx)
```

`GAIN_CURSE_CARD` 修改为：
```gdscript
EventEffectData.EffectType.GAIN_CURSE_CARD:
    var path: String = effect.curse_card_path if effect.curse_card_path != "" else "res://data/cards/xin_mo.tres"
    var curse: CardData = (load(path) as CardData).duplicate()
    player_state.deck.append(curse)
```

`REMOVE_CARDS_TYPE` 和 `get_eligible_cards` 新增 `"诅咒"` 分支：
```gdscript
"诅咒": remove = c.is_curse or c.is_zahuorumuo
```

`_describe_single` 新增：
```gdscript
EventEffectData.EffectType.LOSE_RELIC_RANDOM: return pct + "失去随机遗物"
EventEffectData.EffectType.GAIN_ENERGY_CAP: return pct + "真气上限永久 +%d" % effect.value
EventEffectData.EffectType.REMOVE_CARDS_RANDOM: return pct + "随机移除 %d 张牌" % effect.value
```
