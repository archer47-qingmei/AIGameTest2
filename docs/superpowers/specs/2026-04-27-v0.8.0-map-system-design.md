# v0.8.0 地图系统设计文档

**日期**：2026-04-27  
**项目**：AIGameTest2（Godot 4.6 + GDScript）  
**范围**：v0.8.0 — 引入线性地图界面，2 关战斗，奖励系统正式生效，通关后显示胜利界面

---

## 背景

v0.7.0 建立了奖励选牌系统，但每局只有一场战斗，奖励的牌无处使用。v0.8.0 引入 2 节点线性地图，玩家在两场战斗间经历"奖励 → 地图 → 下一战"的完整 Roguelike 循环，使 v0.7.0 的 PlayerState 架构正式发挥作用。

---

## 架构原则遵守情况

- **MapScreen**：UI 层，只读 GameManager/PlayerState 状态，不含游戏逻辑
- **WinScreen**：UI 层，只做显示和跳转
- **GameManager**：基础设施层，持有 STAGE_ENEMIES 常量和路由逻辑，不含战斗规则
- **CombatEngine**：接受外部 EnemyData 参数，不再硬编码敌人路径

### PlayerState.gd（修改）

```gdscript
class_name PlayerState
extends RefCounted

var deck: Array[CardData] = []
var current_node: int = 0
```

`current_node`：当前待战斗的节点索引（0 = 关1，1 = 关2 Boss）。由 `GameManager.go_to_map()` 递增，`start_new_run()` 时重置为 0。

---

### GameManager.gd（修改）

```gdscript
extends Node

enum Phase { MENU, MAP, COMBAT, REWARD, WIN }

const STAGE_ENEMIES: Array[String] = [
    "res://data/enemies/jaw_worm.tres",
    "res://data/enemies/boss.tres"
]

var current_phase: Phase = Phase.MENU
var player_state: PlayerState

func start_new_run() -> void:
    player_state = PlayerState.new()
    var strike: CardData = preload("res://data/cards/strike.tres")
    var defend: CardData = preload("res://data/cards/defend.tres")
    var bash: CardData   = preload("res://data/cards/bash.tres")
    var slash: CardData  = preload("res://data/cards/slash.tres")
    for i in 4:
        player_state.deck.append(strike.duplicate())
    for i in 4:
        player_state.deck.append(defend.duplicate())
    player_state.deck.append(bash.duplicate())
    player_state.deck.append(slash.duplicate())
    current_phase = Phase.MAP
    get_tree().change_scene_to_file("res://map/MapScreen.tscn")

func go_to_combat() -> void:
    current_phase = Phase.COMBAT
    get_tree().change_scene_to_file("res://combat/CombatScreen.tscn")

func go_to_reward() -> void:
    current_phase = Phase.REWARD
    get_tree().change_scene_to_file("res://reward/RewardScreen.tscn")

func go_to_map() -> void:
    player_state.current_node += 1
    current_phase = Phase.MAP
    get_tree().change_scene_to_file("res://map/MapScreen.tscn")

func go_to_win() -> void:
    current_phase = Phase.WIN
    get_tree().change_scene_to_file("res://win/WinScreen.tscn")

func go_to_menu() -> void:
    player_state = null
    current_phase = Phase.MENU
    get_tree().change_scene_to_file("res://menu/MainMenu.tscn")

func get_current_enemy_data() -> EnemyData:
    return load(STAGE_ENEMIES[player_state.current_node]) as EnemyData

func is_final_node() -> bool:
    return player_state.current_node == STAGE_ENEMIES.size() - 1
```

**注意：** `go_to_combat()` 不再创建 PlayerState（由 `start_new_run()` 负责），可在同一局内多次调用。

### CombatEngine.gd（修改）

`setup()` 新增 `enemy_data: EnemyData` 参数，移除硬编码敌人路径：

```gdscript
func setup(initial_deck: Array[CardData], enemy_data: EnemyData) -> void:
    _enemy_data = enemy_data
    enemy = Combatant.new()
    enemy.display_name = _enemy_data.display_name
    enemy.hp = _enemy_data.hp
    enemy.max_hp = _enemy_data.hp
    enemy.block = 0

    player = Combatant.new()
    player.display_name = "玩家"
    player.hp = 70
    player.max_hp = 70
    player.block = 0

    for card: CardData in initial_deck:
        _draw_pile.append(card.duplicate())
    _draw_pile.shuffle()
    _start_player_turn()
```

### combat/CombatScreen.gd（修改）

`_ready()` 传入敌人数据；`_on_combat_ended()` 按 `is_final_node()` 决定显示哪个按钮：

```gdscript
@onready var _btn_get_reward: Button = $BtnGetReward
@onready var _btn_win: Button        = $BtnWin        # 新增

func _ready() -> void:
    # ... 其他代码不变 ...
    _btn_get_reward.pressed.connect(GameManager.go_to_reward)
    _btn_win.pressed.connect(GameManager.go_to_win)    # 新增
    _engine.setup(GameManager.player_state.deck, GameManager.get_current_enemy_data())  # 修改

func _on_combat_ended(result: String) -> void:
    _lbl_result.text = "胜利！" if result == "victory" else "游戏结束"
    _lbl_result.visible = true
    _btn_end_turn.disabled = true
    if result == "victory":
        if GameManager.is_final_node():
            _btn_win.visible = true        # 新增：最终节点显示通关按钮
        else:
            _btn_get_reward.visible = true
    else:
        _btn_return_menu.visible = true
    for btn: Button in _hand_buttons:
        btn.disabled = true
```

`CombatScreen.tscn` 新增 `BtnWin` 节点（Button，visible=false，text="通关！"，锚点居中偏下）。

### map/MapScreen.gd（新增）

```gdscript
extends Control

const NODE_LABELS: Array[String] = ["关卡一：颚虫", "关卡二：精英守卫"]

func _ready() -> void:
    var current: int = GameManager.player_state.current_node
    for i in NODE_LABELS.size():
        var btn := Button.new()
        if i < current:
            btn.text = NODE_LABELS[i] + "（已完成）"
            btn.disabled = true
        elif i == current:
            btn.text = NODE_LABELS[i]
            btn.pressed.connect(GameManager.go_to_combat)
        else:
            btn.text = NODE_LABELS[i] + "（未解锁）"
            btn.disabled = true
        $NodeContainer.add_child(btn)
```

`MapScreen.tscn`：Control 根节点 + Label 标题（"选择关卡"）+ VBoxContainer（NodeContainer，动态填入节点按钮）。

## 游戏流程

```
主菜单
  └─→ 点击"开始游戏" → start_new_run()
        PlayerState.current_node = 0
        └─→ MapScreen
              关卡一[可点击] → 关卡二[锁定]
              └─→ 点击关卡一 → go_to_combat()
                    └─→ CombatScreen（颚虫）
                          ├─ 胜利（非终点）→ "领取奖励" → go_to_reward()
                          │     └─→ RewardScreen → 选牌 → go_to_map()
                          │           current_node = 1
                          │           └─→ MapScreen
                          │                 关卡一[完成] → 关卡二[可点击]
                          │                 └─→ 点击关卡二 → go_to_combat()
                          │                       └─→ CombatScreen（精英守卫）
                          │                             ├─ 胜利（终点）→ "通关！" → go_to_win()
                          │                             │     └─→ WinScreen → 返回主菜单
                          │                             └─ 败北 → go_to_menu() → 主菜单
                          └─ 败北 → go_to_menu() → 主菜单
```
