# v0.13.0 状态效果（虚弱）设计文档

**日期：** 2026-04-28
**目标：** 战斗中新增"虚弱"状态效果，玩家可通过新卡牌施加给敌人，使其攻击伤害降低 25%。

---

## 核心决策

| 问题 | 决策 |
|---|---|
| 状态效果种类 | 仅虚弱（weak），本版不做其他效果 |
| 计时方式 | 按回合递减：每回合结束 -1，降至 0 时失效 |
| 伤害公式 | `int(dmg * 0.75)`（施害方 weak > 0 时） |
| 敌方虚弱消耗时机 | 敌人执行攻击后递减（不在格挡回合递减） |
| 玩家虚弱消耗时机 | 玩家回合开始时递减（本版玩家不会被施加虚弱，预留接口） |
| 新卡牌 | 1 张：缠绕（1 费，施加 2 层虚弱给敌人） |
| 玩家面板显示 | 不显示玩家虚弱（本版无来源，保持界面简洁） |

---

## 架构总览

```
逻辑层
  Combatant.gd       新增 weak 字段 + add_weak() 方法
  EffectResolver.gd  新增 "weak" 效果类型处理；伤害计算加虚弱修正
  CombatEngine.gd    敌方回合施加虚弱修正并递减；玩家回合开始递减

UI 层
  combat/CombatScreen.gd/.tscn   新增 LblEnemyWeak，_refresh_ui() 更新

数据层
  data/cards/entangle.tres       新卡牌文件
```

**数据流：**

```
玩家打出"缠绕"
  → CombatEngine.play_card() → EffectResolver.apply_effects()
  → effect.type == "weak" → target.add_weak(2)
  → state_changed 信号 → CombatScreen._refresh_ui()
  → LblEnemyWeak 显示 "虚弱：2"

敌人回合（攻击）：
  → dmg = action.value
  → if enemy.weak > 0: dmg = int(dmg * 0.75)
  → apply damage to player
  → enemy.weak = max(0, enemy.weak - 1)

玩家新回合开始：
  → player.weak = max(0, player.weak - 1)  （预留，本版恒为 0）
```

---

## 逻辑层

### Combatant.gd

新增字段与方法：

```gdscript
var weak: int = 0

func add_weak(n: int) -> void:
    weak += n
```

其余代码不动。

---

### EffectResolver.gd

**新增 "weak" 效果类型：**

在 `apply_effects()` 的 match 块中新增：

```gdscript
"weak":
    target.add_weak(effect.value)
```

**伤害计算加虚弱修正：**

`apply_damage()` 方法签名新增 `source: Combatant` 参数，伤害计算前检查：

```gdscript
static func apply_damage(source: Combatant, target: Combatant, amount: int) -> void:
    var dmg: int = amount
    if source.weak > 0:
        dmg = int(dmg * 0.75)
    # 原有 block/hp 计算逻辑不变
```

`apply_effects()` 中调用 `apply_damage` 时传入 `source`。

---

### CombatEngine.gd

**`_do_enemy_turn()`：**

```gdscript
func _do_enemy_turn() -> void:
    var action: EnemyActionData = get_current_enemy_action()
    if action.type == "attack":
        EffectResolver.apply_damage(enemy, player, action.value)
        enemy.weak = max(0, enemy.weak - 1)
    elif action.type == "block":
        enemy.block += action.value
    _advance_enemy_action()
    state_changed.emit()
```

**`_start_player_turn()`：**

```gdscript
func _start_player_turn() -> void:
    player.weak = max(0, player.weak - 1)
    # 其余原有逻辑不变（补充能量、抽牌等）
```

新增 `@onready`：

```gdscript
@onready var _lbl_enemy_weak: Label = $VBoxContainer/EnemyPanel/LblEnemyWeak
```

```gdscript
_lbl_enemy_weak.text = "虚弱：%d" % _engine.enemy.weak
```
