# v0.19.0 连线地图 设计文档

**日期：** 2026-04-29
**目标：** 将线性节点序列升级为分支图结构，MapScreen 展示节点按钮与连线，玩家可选择路径。

---

## 核心决策

| 问题 | 决策 |
|---|---|
| 图结构 vs 线性 | 图结构：NodeData 持有 `connections: Array[NodeData]` |
| 固定 vs 随机 | 本版本固定结构（2→2→1），随机化作为独立后续版本 |
| 架构方案 | 方案 A：扩展现有文件，不新增逻辑类 |
| 导航状态存放 | PlayerState（纯数据字段），GameManager 处理场景跳转 |
| 地图信息展示 | 只显示节点类型（战斗/休息/Boss）和连线，不显示 HP 或敌人名 |

---

## 固定图结构

```
col=0           col=1           col=2
[a1 战斗]──┬──[b1 休息]──┐
           │              ├──[c1 Boss]
[a2 战斗]──┴──[b2 战斗]──┘
```

连线：a1→b1, a1→b2, a2→b1, a2→b2, b1→c1, b2→c1。`connections` 为空 = Boss（终点）。

## 各文件详细设计

### NodeData.gd（修改）

新增字段：

```gdscript
var connections: Array[NodeData] = []
var col: int = 0
var row: int = 0
```

`col`/`row` 仅供 MapScreen 计算屏幕坐标，不含导航语义。`connections` 为空表示终点。

---

### PlayerState.gd（修改）

移除：
```gdscript
var current_node: int = 0
var node_sequence: Array[NodeData] = []
```

新增：
```gdscript
var map_all_nodes: Array[NodeData] = []
var available_nodes: Array[NodeData] = []
var completed_nodes: Array[NodeData] = []
var current_node: NodeData = null
```

`map_all_nodes` 由 `start_new_run()` 一次性写入后只读。`available_nodes`/`completed_nodes` 由 `go_to_map()` 更新。`current_node` 在 `select_node()` 时设置，`go_to_map()` 时清空。

---

### GameManager.gd（修改）

**`start_new_run()` 图构建：**

```gdscript
var a1: NodeData = NodeData.new()
a1.type = NodeData.Type.COMBAT; a1.enemy_data = load(NORMAL_POOL[0]) as EnemyData
a1.col = 0; a1.row = 0

var a2: NodeData = NodeData.new()
a2.type = NodeData.Type.COMBAT; a2.enemy_data = load(NORMAL_POOL[1]) as EnemyData
a2.col = 0; a2.row = 1

var b1: NodeData = NodeData.new()
b1.type = NodeData.Type.REST
b1.col = 1; b1.row = 0

var b2: NodeData = NodeData.new()
b2.type = NodeData.Type.COMBAT; b2.enemy_data = load(NORMAL_POOL[0]) as EnemyData
b2.col = 1; b2.row = 1

var c1: NodeData = NodeData.new()
c1.type = NodeData.Type.COMBAT; c1.enemy_data = load(BOSS_ENEMY) as EnemyData
c1.col = 2; c1.row = 0

a1.connections.assign([b1, b2])
a2.connections.assign([b1, b2])
b1.connections.assign([c1])
b2.connections.assign([c1])
# c1.connections 保持空数组

player_state.map_all_nodes.assign([a1, a2, b1, b2, c1])
player_state.available_nodes.assign([a1, a2])
```

**`select_node(node: NodeData)`（新增）：**

```gdscript
func select_node(node: NodeData) -> void:
    player_state.current_node = node
    if node.type == NodeData.Type.REST:
        go_to_rest()
    else:
        go_to_combat()
```

**`go_to_map()`（修改）：**

```gdscript
func go_to_map() -> void:
    if player_state.current_node != null:
        player_state.completed_nodes.append(player_state.current_node)
        player_state.available_nodes.assign(player_state.current_node.connections)
        player_state.current_node = null
    current_phase = Phase.MAP
    get_tree().change_scene_to_file("res://map/MapScreen.tscn")
```

**`get_current_enemy_data()`（修改）：**

```gdscript
func get_current_enemy_data() -> EnemyData:
    return player_state.current_node.enemy_data
```

**`is_final_node()`（修改）：**

```gdscript
func is_final_node() -> bool:
    return player_state.current_node.connections.is_empty()
```

### map/MapScreen.gd（全部重写）

节点坐标（绝对坐标，480×854 画布）：

| 位置 | x | y |
|---|---|---|
| col=0, row=0 | 100 | 250 |
| col=0, row=1 | 100 | 500 |
| col=1, row=0 | 240 | 250 |
| col=1, row=1 | 240 | 500 |
| col=2（Boss） | 380 | 375 |

按钮大小：110×50，position = center - (55, 25)。

按钮状态规则：
- `available_nodes` 中：可点击，`btn.pressed.connect(GameManager.select_node.bind(nd))`
- `completed_nodes` 中：禁用，文字前缀 `✓ `
- 其余（locked）：禁用，无前缀

连线颜色规则（`_draw()` 中）：
- 出发节点在 `completed_nodes` 中：绿色 `Color(0.4, 1.0, 0.4)`
- 否则：灰色 `Color(0.5, 0.5, 0.5)`

节点文字规则：
- `connections.is_empty()` → `"Boss 战斗"`
- `type == REST` → `"休息站"`
- 否则 → `"战斗"`

```gdscript
extends Control

@onready var _lbl_hp: Label = $LblHP

var _node_positions: Dictionary = {}

func _ready() -> void:
    var state: PlayerState = GameManager.player_state
    _lbl_hp.text = "生命：%d / %d" % [state.hp, state.max_hp]
    _build_map(state)
    queue_redraw()

func _build_map(state: PlayerState) -> void:
    for nd: NodeData in state.map_all_nodes:
        var center: Vector2 = _get_node_pos(nd)
        _node_positions[nd] = center
        var btn := Button.new()
        btn.size = Vector2(110.0, 50.0)
        btn.position = center - Vector2(55.0, 25.0)
        var is_available: bool = state.available_nodes.has(nd)
        var is_completed: bool = state.completed_nodes.has(nd)
        if is_completed:
            btn.text = "✓ " + _get_node_label(nd)
            btn.disabled = true
        elif is_available:
            btn.text = _get_node_label(nd)
            btn.pressed.connect(GameManager.select_node.bind(nd))
        else:
            btn.text = _get_node_label(nd)
            btn.disabled = true
        add_child(btn)

func _draw() -> void:
    var state: PlayerState = GameManager.player_state
    for nd: NodeData in state.map_all_nodes:
        if not _node_positions.has(nd):
            continue
        var from: Vector2 = _node_positions[nd]
        for target: NodeData in nd.connections:
            if not _node_positions.has(target):
                continue
            var to: Vector2 = _node_positions[target]
            var color: Color = Color(0.4, 1.0, 0.4) if state.completed_nodes.has(nd) \
                               else Color(0.5, 0.5, 0.5)
            draw_line(from, to, color, 2.0)

func _get_node_pos(nd: NodeData) -> Vector2:
    if nd.col == 2:
        return Vector2(380.0, 375.0)
    return Vector2(100.0 if nd.col == 0 else 240.0,
                   250.0 if nd.row == 0 else 500.0)

func _get_node_label(nd: NodeData) -> String:
    if nd.connections.is_empty():
        return "Boss 战斗"
    if nd.type == NodeData.Type.REST:
        return "休息站"
    return "战斗"
```
