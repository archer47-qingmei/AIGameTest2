# v0.3.0 数据文件设计文档

**日期**：2026-04-24  
**项目**：AIGameTest2（Godot 4.6 + GDScript）  
**范围**：v0.3.0 — 引入 `.tres` 数据文件替换 CombatEngine 硬编码数据（纯重构，零功能变化）

---

## 背景

v0.2.0 完成了逻辑层与 UI 层的分离。但 `CombatEngine.setup()` 内仍硬编码了卡牌数值和敌人属性，每次新增卡牌或敌人都需要修改逻辑代码。

v0.3.0 目标：将卡牌和敌人数据外置为 `.tres` 资源文件，`CombatEngine` 只负责加载和运行，不再持有任何内容数值。

---

## 新增 Resource 类

### EnemyActionData.gd

```gdscript
class_name EnemyActionData
extends Resource

@export var type: String   # "attack" 或 "block"
@export var value: int
```

### EnemyData.gd

```gdscript
class_name EnemyData
extends Resource

@export var display_name: String
@export var hp: int
@export var actions: Array[EnemyActionData]
```

**职责边界：**
- `EnemyData` = 静态蓝图（存文件，只读）
- `Combatant` = 运行时实例（战斗中变化的 HP、block）
- 两者完全分离，支持同类型敌人多实例

## CombatEngine 改动

### 新增私有变量

```gdscript
var _enemy_data: EnemyData
```

### 改写 setup()

```gdscript
func setup() -> void:
    # 加载敌人数据
    _enemy_data = preload("res://data/enemies/jaw_worm.tres")
    enemy = Combatant.new()
    enemy.display_name = _enemy_data.display_name
    enemy.hp = _enemy_data.hp
    enemy.max_hp = _enemy_data.hp
    enemy.block = 0

    # 玩家（不变）
    player = Combatant.new()
    player.display_name = "玩家"
    player.hp = 70
    player.max_hp = 70
    player.block = 0

    # 从文件加载牌库，每张必须 duplicate() 确保独立实例
    var strike: CardData = preload("res://data/cards/strike.tres")
    var defend: CardData = preload("res://data/cards/defend.tres")
    var bash:   CardData = preload("res://data/cards/bash.tres")
    for i in 4:
        _draw_pile.append(strike.duplicate())
    for i in 4:
        _draw_pile.append(defend.duplicate())
    _draw_pile.append(bash.duplicate())

    _draw_pile.shuffle()
    _start_player_turn()
```

> **注意：** `preload()` 返回同一共享实例，必须调用 `.duplicate()` 才能获得独立副本，否则修改一张牌会影响所有同名牌。

### 新增公开方法

```gdscript
func get_current_enemy_action() -> EnemyActionData:
    return _enemy_data.actions[(turn_number - 1) % _enemy_data.actions.size()]
```

### 改写 _do_enemy_turn()

```gdscript
func _do_enemy_turn() -> void:
    enemy.block = 0
    var action: EnemyActionData = _enemy_data.actions[(turn_number - 1) % _enemy_data.actions.size()]
    if action.type == "attack":
        player.take_damage(action.value)
    else:
        enemy.add_block(action.value)
```

## CombatScreen 改动

`_refresh_ui()` 中意图显示改为读取引擎数据，替换原来的硬编码判断：

```gdscript
# 旧：
var intent: String = "攻击 11" if _engine.turn_number % 2 == 1 else "格挡 6"

# 新：
var action: EnemyActionData = _engine.get_current_enemy_action()
var intent: String = "攻击 %d" % action.value if action.type == "attack" else "格挡 %d" % action.value
```
