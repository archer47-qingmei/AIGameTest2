# v0.49.0 彩色拖拽连线 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将拖拽出牌时的白色虚线改为彩色——攻击牌（single/all）对敌人显示红线，自身牌（none）对玩家面板显示绿线。

**Architecture:** 在 DragLayer.gd 新增两个颜色常量和 `_player_local_pos` 字段，`begin_drag()` 加一个 `player_global_pos` 参数，`_draw()` 按 `_target_type` 选颜色；CombatScreen.gd 调用时多传一个玩家面板中心坐标。

**Tech Stack:** Godot 4.4 GDScript，严格类型，CanvasItem._draw()，`get_global_transform().affine_inverse()`。

本任务无自动化测试框架，验证方式为启动 Godot 并手动测试（见步骤末尾）。

- [ ] **Step 1: 在常量区末尾添加两个颜色常量**

打开 `combat/DragLayer.gd`。在第 17 行 `const LINE_DASH := 12.0` 之后，紧接着插入：

```gdscript
const LINE_COLOR_ENEMY := Color(1.0, 0.3, 0.3, 0.85)
const LINE_COLOR_SELF  := Color(0.3, 1.0, 0.4, 0.85)
```

- [ ] **Step 2: 在字段区末尾添加 `_player_local_pos`**

在第 28 行 `var _target_line_ends: Array[Vector2] = []` 之后插入：

```gdscript
var _player_local_pos: Vector2 = Vector2.ZERO
```

- [ ] **Step 3: 更新 `begin_drag()` 签名和函数体**

将第 30–51 行的 `begin_drag()` 完整替换为：

```gdscript
func begin_drag(card_index: int, card_global_pos: Vector2, card_text: String,
		target_type: String, enemy_global_positions: Array[Vector2],
		enemy_engine_indices: Array[int], player_global_pos: Vector2) -> void:
	_drag_gen += 1
	if _ghost_card != null:
		_ghost_card.queue_free()
		_ghost_card = null
	_card_index = card_index
	_target_type = target_type
	_origin_local_pos = get_global_transform().affine_inverse() * card_global_pos
	_enemy_engine_indices = enemy_engine_indices.duplicate()
	_enemy_local_positions.clear()
	for gp in enemy_global_positions:
		_enemy_local_positions.append(get_global_transform().affine_inverse() * gp)
	_player_local_pos = get_global_transform().affine_inverse() * player_global_pos
	_current_slot = -1
	_state = State.LIFTED
	mouse_filter = MOUSE_FILTER_STOP
	_create_ghost(card_text, _origin_local_pos)
	if target_type == TARGET_ALL:
		_target_line_ends = _enemy_local_positions.duplicate()
		if not _enemy_local_positions.is_empty():
			_current_slot = 0
	elif target_type == TARGET_NONE:
		_target_line_ends = [_player_local_pos]
```

- [ ] **Step 4: 更新 `_draw()` 使用彩色**

将第 179–184 行的 `_draw()` 完整替换为：

```gdscript
func _draw() -> void:
	if _state != State.DRAGGING or _ghost_card == null:
		return
	var from := _ghost_card.position + Vector2(GHOST_SIZE.x / 2.0, 0.0)
	var color := LINE_COLOR_SELF if _target_type == TARGET_NONE else LINE_COLOR_ENEMY
	for target_pos: Vector2 in _target_line_ends:
		draw_dashed_line(from, target_pos, color, LINE_WIDTH, LINE_DASH)
```

- [ ] **Step 5: 更新 `_cleanup()` 重置 `_player_local_pos`**

在 `_cleanup()` 函数体中，在 `_current_slot = -1` 这一行之后加一行：

```gdscript
	_player_local_pos = Vector2.ZERO
```

完整的 `_cleanup()` 应为：

```gdscript
func _cleanup() -> void:
	if _ghost_card != null:
		_ghost_card.queue_free()
		_ghost_card = null
	_state = State.IDLE
	_card_index = -1
	_target_type = ""
	_enemy_local_positions.clear()
	_enemy_engine_indices.clear()
	_target_line_ends.clear()
	_current_slot = -1
	_player_local_pos = Vector2.ZERO
	mouse_filter = MOUSE_FILTER_IGNORE
	queue_redraw()
```

## 手动验收

在 Godot 编辑器中运行项目，进入战斗场景：

1. 拖动一张攻击牌（如「斩击」，target_type=single）→ 虚线应为**红色**，指向最近敌人
2. 拖动一张 AoE 牌（如「横扫」，target_type=all）→ 所有连线均为**红色**
3. 拖动一张自身牌（如「防御」或「侧步」，target_type=none）→ 虚线应为**绿色**，指向玩家面板
4. 拖动任意牌后取消（拖回手牌区松手）→ 连线消失，无颜色残留
5. 主菜单版本号显示 **v0.49.0**
