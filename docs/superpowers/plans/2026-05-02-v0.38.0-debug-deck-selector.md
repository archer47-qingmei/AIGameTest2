# v0.38.0 调试牌组选择器 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在角色选择界面新增"测试模式"按钮，打开调试牌组选择器，允许开发者自由勾选卡牌（支持多份）后以空牌组直接开始跑图。

**Architecture:** 新增独立场景 `debug/DebugDeckScreen`，负责展示 21 张剑修奖励牌并提供 +/- 数量控制。`GameManager` 新增 `go_to_debug_deck()` 和 `start_debug_run(cards)` 两个方法；`CharSelectScreen` 追加一个"测试模式"按钮。`go_to_char_select()` 已存在于 `GameManager.gd:118`，无需新增。

**Tech Stack:** Godot 4 GDScript，`.tscn` 场景文件，`CardPool.SWORD_REWARD_CARDS`（21 条路径）

- [ ] **Step 1: 在 `go_to_char_select()` 之后插入两个新方法**

打开 `GameManager.gd`，在第 121 行（`go_to_char_select` 方法末尾的空行）之后插入：

```gdscript
func go_to_debug_deck() -> void:
	current_phase = Phase.CHAR_SELECT
	get_tree().change_scene_to_file("res://debug/DebugDeckScreen.tscn")

func start_debug_run(cards: Array[CardData]) -> void:
	pending_card_reward = true
	player_state = PlayerState.new()
	player_state.character = "sword"
	for path in CardPool.SWORD_START_RELICS:
		player_state.relics.append((load(path) as RelicData).duplicate())
	for card in cards:
		player_state.deck.append(card.duplicate())
	var nodes: Array[NodeData] = MapGenerator.generate()
	player_state.map_all_nodes.assign(nodes)
	var start_nodes: Array[NodeData] = []
	for nd in nodes:
		if nd.config.column == 0:
			start_nodes.append(nd)
	player_state.available_nodes.assign(start_nodes)
	current_phase = Phase.MAP
	get_tree().change_scene_to_file("res://map/MapScreen.tscn")
```

> `start_debug_run` 与 `start_new_run` 唯一的区别：跳过硬编码起始牌组，改用传入的 `cards`。

## Task 3: 创建 DebugDeckScreen

**Files:**
- Create: `debug/DebugDeckScreen.gd`
- Create: `debug/DebugDeckScreen.tscn`

- [ ] **Step 1: 创建 `debug/DebugDeckScreen.gd`**

新建文件 `debug/DebugDeckScreen.gd`，内容如下：

```gdscript
extends Control

@onready var _card_list: VBoxContainer = $VBoxContainer/ScrollContainer/CardListContainer
@onready var _lbl_selected: Label = $VBoxContainer/LblSelected
@onready var _btn_start: Button = $VBoxContainer/BtnRow/BtnStart
@onready var _btn_back: Button = $VBoxContainer/BtnRow/BtnBack

var _cards: Array[CardData] = []
var _counts: Array[int] = []

func _ready() -> void:
	for path in CardPool.SWORD_REWARD_CARDS:
		_cards.append(load(path) as CardData)
		_counts.append(0)
	_build_rows()
	_btn_start.pressed.connect(_on_start_pressed)
	_btn_back.pressed.connect(_on_back_pressed)
	_refresh()

func _build_rows() -> void:
	for i in _cards.size():
		var row := HBoxContainer.new()
		var lbl_name := Label.new()
		lbl_name.text = _cards[i].card_name
		lbl_name.custom_minimum_size.x = 80
		var lbl_desc := Label.new()
		lbl_desc.text = _cards[i].get_description()
		lbl_desc.custom_minimum_size.x = 280
		lbl_desc.autowrap_mode = TextServer.AUTOWRAP_WORD_SMART
		var btn_minus := Button.new()
		btn_minus.text = "−"
		btn_minus.custom_minimum_size.x = 30
		btn_minus.pressed.connect(_on_minus.bind(i))
		var lbl_count := Label.new()
		lbl_count.text = "0"
		lbl_count.custom_minimum_size.x = 30
		lbl_count.horizontal_alignment = HORIZONTAL_ALIGNMENT_CENTER
		var btn_plus := Button.new()
		btn_plus.text = "+"
		btn_plus.custom_minimum_size.x = 30
		btn_plus.pressed.connect(_on_plus.bind(i))
		row.add_child(lbl_name)
		row.add_child(lbl_desc)
		row.add_child(btn_minus)
		row.add_child(lbl_count)
		row.add_child(btn_plus)
		_card_list.add_child(row)

func _on_minus(i: int) -> void:
	_counts[i] = max(0, _counts[i] - 1)
	_refresh()

func _on_plus(i: int) -> void:
	_counts[i] = min(5, _counts[i] + 1)
	_refresh()

func _refresh() -> void:
	var total := 0
	for i in _cards.size():
		total += _counts[i]
		var row: HBoxContainer = _card_list.get_child(i) as HBoxContainer
		(row.get_child(3) as Label).text = str(_counts[i])
	_lbl_selected.text = "已选：%d 张" % total
	_btn_start.disabled = total == 0

func _on_start_pressed() -> void:
	var cards: Array[CardData] = []
	for i in _cards.size():
		for _j in _counts[i]:
			cards.append(_cards[i])
	GameManager.start_debug_run(cards)

func _on_back_pressed() -> void:
	GameManager.go_to_char_select()
```

- [ ] **Step 3: 手动验证**

运行游戏，按以下路径验收：

1. 主菜单 → 选择职业：出现"测试模式"按钮
2. 点击"测试模式"：进入调试牌组界面，显示 21 行卡牌，每行有牌名、效果、`−` `0` `+`
3. 所有数量为 0 时："开始测试"按钮禁用
4. 点击某张牌的 `+`：数量变 1，底部显示"已选：1 张"，"开始测试"按钮启用
5. 连续点 `+` 至 5：数量停在 5（不超过）
6. 点 `−` 至 0：数量停在 0（不低于）
7. 选几张牌后点"开始测试"：进入地图界面
8. 进入战斗，确认初始手牌只包含所选卡牌（无战铁/侧步等默认牌）
9. 点"返回"：回到角色选择界面
