## description 字段格式说明

所有遗物 description 字段采用统一格式：

```
[文学描述段落]\n\n[效果说明]
```

- 文学描述：留白风格，只写感官印象，不解释效果
- `\n\n` 在 .tres 字符串里表示空行，Godot Label 渲染时会换两行
- 效果说明：简洁规则描述，让玩家一眼看懂效果
