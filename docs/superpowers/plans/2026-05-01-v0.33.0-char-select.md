# v0.33.0 职业选择 + 剑修身份 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在主菜单和地图之间插入职业选择界面，玩家选择剑修后携带本命法宝「养剑葫」开始游戏（剑意效果暂时静默）。

**Architecture:** 新建 CharSelectScreen 场景对应 GameManager.Phase.CHAR_SELECT；PlayerState 记录 character 字段；养剑葫通过新增 EffectType.SWORD_INTENT = 4 占位，RelicEngine 无对应 match 分支，静默无效果；CardPool.SWORD_START_RELICS 集中管理剑修起始遗物路径。

**Tech Stack:** Godot 4 GDScript, Resource (.tres), GDScript signal connections

### Task 1: EffectType.SWORD_INTENT 枚举占位符

**Files:**
- Modify: `RelicData.gd`

- [ ] **Step 1: 修改 RelicData.gd，在 EffectType 枚举末尾追加 SWORD_INTENT**

将 `RelicData.gd` 中的：
```gdscript
enum EffectType { ENERGY, HEAL_HP, BLOCK, DRAW }
```

改为：
```gdscript
enum EffectType { ENERGY, HEAL_HP, BLOCK, DRAW, SWORD_INTENT }
```

- [ ] **Step 2: 确认 RelicEngine.gd 无需修改**

打开 `RelicEngine.gd`，确认 `_apply()` 函数的 match 语句无 SWORD_INTENT 分支：

```gdscript
static func _apply(relic: RelicData, engine: CombatEngine) -> void:
	match relic.effect_type:
		RelicData.EffectType.ENERGY:
			engine.energy += relic.value
		RelicData.EffectType.HEAL_HP:
			engine.player.hp = mini(engine.player.hp + relic.value, engine.player.max_hp)
		RelicData.EffectType.BLOCK:
			engine.player.add_block(relic.value)
		RelicData.EffectType.DRAW:
			engine.draw_cards(relic.value)
```

GDScript match 语句无匹配分支时静默跳过，无需修改 RelicEngine.gd。

- [ ] **Step 3: 提交**

```bash
git add RelicData.gd
git commit -m "feat: add SWORD_INTENT placeholder to RelicData EffectType"
```

- [ ] **Step 3: 将 start_new_run() 替换为接收 character 参数的新版本**

将现有的 `func start_new_run() -> void:` 方法完整替换为：

```gdscript
func start_new_run(character: String) -> void:
	pending_card_reward = true
	player_state = PlayerState.new()
	player_state.character = character
	var relic_paths: Array[String] = []
	match character:
		"sword":
			relic_paths.assign(CardPool.SWORD_START_RELICS)
	for path in relic_paths:
		player_state.relics.append((load(path) as RelicData).duplicate())

	var strike: CardData    = preload("res://data/cards/strike.tres")
	var defend: CardData    = preload("res://data/cards/defend.tres")
	var bash: CardData      = preload("res://data/cards/bash.tres")
	var slash: CardData     = preload("res://data/cards/slash.tres")
	var whirlwind: CardData = preload("res://data/cards/whirlwind.tres")
	for i in 4:
		player_state.deck.append(strike.duplicate())
	for i in 4:
		player_state.deck.append(defend.duplicate())
	player_state.deck.append(bash.duplicate())
	player_state.deck.append(slash.duplicate())
	player_state.deck.append(whirlwind.duplicate())

	var nodes: Array[NodeData] = MapGenerator.generate()
	player_state.map_all_nodes.assign(nodes)
	var start_nodes: Array[NodeData] = []
	for nd in nodes:
		if nd.config.column == 0:
			start_nodes.append(nd)
	player_state.available_nodes.assign(start_nodes)

	current_phase = Phase.MAP
	get_tree().change_scene_to_file("res://map/MapScreen.tscn")
```
