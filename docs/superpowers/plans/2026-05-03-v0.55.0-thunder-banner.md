# v0.55.0 雷蛙与阵旗 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增雷蛙和阵旗两只普通怪物，配套引入敌人力量（strength）机制、纯视觉蓄力（pre_charge）和群体增益（group_strengthen / group_block）三个新机制。

**Architecture:** 分四层：①Combatant 加 strength 字段 + EffectResolver 集成；②Engine/Screen 扩展三个新 action type；③两只怪物的 .tres 数据文件；④阵旗组合 group 文件 + MapGenerator 注册。strength 在 apply_damage() 里以 `amount + source.strength` 方式加入，只影响敌人攻击路径。

**Tech Stack:** GDScript 4，Godot 4.4 .tres resource 文件

### Task 2: Engine + Screen 扩展三个新 action type

新增：
- `pre_charge`：纯视觉蓄力，`_do_enemy_turn()` 中什么都不做，意图显示"意图：蓄力"
- `group_strengthen`：所有存活敌人 +N strength
- `group_block`：所有存活敌人 +N block

同时修复 `_intent_text()` 以将 `e.strength` 计入攻击意图显示值。

**Files:**
- Modify: `CombatEngine.gd`
- Modify: `combat/CombatScreen.gd`

- [ ] **Step 1: 修改 `CombatEngine.gd` — `_do_enemy_turn()` 新增三个分支**

找到 `_do_enemy_turn()` 中的 `"charge":` 分支（第 271-273 行）：
```gdscript
			"charge":
				enemies[i].is_charging = true
				enemies[i].charge_value = action.value
```

在这段之后（`"poison":` 之前）插入三个新分支：
```gdscript
			"pre_charge":
				pass
			"group_strengthen":
				for j in enemies.size():
					if enemies[j].hp > 0:
						enemies[j].strength += action.value
			"group_block":
				for j in enemies.size():
					if enemies[j].hp > 0:
						enemies[j].add_block(action.value)
```

插入后该区域完整内容：
```gdscript
			"charge":
				enemies[i].is_charging = true
				enemies[i].charge_value = action.value
			"pre_charge":
				pass
			"group_strengthen":
				for j in enemies.size():
					if enemies[j].hp > 0:
						enemies[j].strength += action.value
			"group_block":
				for j in enemies.size():
					if enemies[j].hp > 0:
						enemies[j].add_block(action.value)
			"poison":
				for _j in action.value:
					_draw_pile.append(VENOM_CARD.duplicate())
				_draw_pile.shuffle()
```

- [ ] **Step 2: 修改 `combat/CombatScreen.gd` — `_intent_text()` 加 strength + 三个新 case**

当前方法（第 131-151 行）：
```gdscript
func _intent_text(action: EnemyActionData, e: Combatant) -> String:
	var val: int = int(action.value * 0.75) if e.weak > 0 else action.value
	match action.type:
		"attack":
			return "意图：攻击 %d" % val
		"charge_attack":
			return "意图：蓄力攻击 %d" % val
		"attack_weak":
			return "意图：攻击 %d + 虚弱" % val
		"attack_vulnerable":
			return "意图：攻击 %d + 易伤" % val
		"multi_attack":
			return "意图：连斩 %d ×%d" % [val, action.count]
		"charge":
			return "意图：蓄力"
		"poison":
			return "意图：投毒 %d" % action.value
		"discard_curse":
			return "意图：心魔 ×%d" % action.value
		_:
			return "意图：格挡 %d" % action.value
```

替换为：
```gdscript
func _intent_text(action: EnemyActionData, e: Combatant) -> String:
	var base: int = action.value + e.strength
	var val: int = int(base * 0.75) if e.weak > 0 else base
	match action.type:
		"attack":
			return "意图：攻击 %d" % val
		"charge_attack":
			return "意图：蓄力攻击 %d" % val
		"attack_weak":
			return "意图：攻击 %d + 虚弱" % val
		"attack_vulnerable":
			return "意图：攻击 %d + 易伤" % val
		"multi_attack":
			return "意图：连斩 %d ×%d" % [val, action.count]
		"charge":
			return "意图：蓄力"
		"pre_charge":
			return "意图：蓄力"
		"group_strengthen":
			return "意图：强化 +%d力量" % action.value
		"group_block":
			return "意图：守护 +%d格挡" % action.value
		"poison":
			return "意图：投毒 %d" % action.value
		"discard_curse":
			return "意图：心魔 ×%d" % action.value
		_:
			return "意图：格挡 %d" % action.value
```
