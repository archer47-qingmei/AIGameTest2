# v0.47.0 手牌扇形布局 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将手牌区从 ScrollContainer+HBoxContainer 改为 Control 绝对定位布局，所有牌始终可见、牌多时叠压展开，不依赖滚动。

**Architecture:** 用单个 `Control`（HandArea）替换 `ScrollContainer/HandContainer`，在 `_rebuild_hand()` 中创建按钮时设置绝对 size，新增 `_layout_hand()` 根据容器宽度动态计算每张牌的 x 坐标，连接 `resized` 信号处理尺寸变化。

**Tech Stack:** Godot 4 GDScript，严格类型，Control 绝对坐标定位。

### Task 1: 替换 tscn 中的手牌容器节点

**Files:**
- Modify: `combat/CombatScreen.tscn`

- [ ] **Step 1: 找到并替换 HandScroll + HandContainer 节点**

在 `combat/CombatScreen.tscn` 中找到以下两个节点（当前在文件第 119-124 行附近）：

```
[node name="HandScroll" type="ScrollContainer" parent="VBoxContainer"]
custom_minimum_size = Vector2(0, 170)
layout_mode = 2

[node name="HandContainer" type="HBoxContainer" parent="VBoxContainer/HandScroll"]
layout_mode = 2
```

将这两个节点**整体替换**为：

```
[node name="HandArea" type="Control" parent="VBoxContainer"]
custom_minimum_size = Vector2(0, 170)
layout_mode = 2
clip_contents = true
```

注意：`clip_contents = true` 防止叠压的卡牌绘制到容器外面。

- [ ] **Step 3: 改写 _rebuild_hand()**

找到当前的 `_rebuild_hand()`（第 159-174 行）：

```gdscript
func _rebuild_hand() -> void:
	for btn: Button in _hand_buttons:
		btn.queue_free()
	_hand_buttons.clear()
	for i in _engine.hand.size():
		var card: CardData = _engine.hand[i]
		var btn: Button = Button.new()
		btn.custom_minimum_size = Vector2(110, 150)
		btn.autowrap_mode = TextServer.AUTOWRAP_WORD_SMART
		btn.vertical_alignment = VERTICAL_ALIGNMENT_TOP
		btn.text = card.get_description()
		btn.pressed.connect(_on_card_pressed.bind(i))
		_hand_container.add_child(btn)
		_hand_buttons.append(btn)
	if _pending_card_index >= 0:
		_set_targeting_mode(true)
```

替换为：

```gdscript
func _rebuild_hand() -> void:
	for btn: Button in _hand_buttons:
		btn.queue_free()
	_hand_buttons.clear()
	for i in _engine.hand.size():
		var card: CardData = _engine.hand[i]
		var btn := Button.new()
		btn.custom_minimum_size = Vector2(110, 150)
		btn.size = Vector2(110, 150)
		btn.autowrap_mode = TextServer.AUTOWRAP_WORD_SMART
		btn.vertical_alignment = VERTICAL_ALIGNMENT_TOP
		btn.text = card.get_description()
		btn.pressed.connect(_on_card_pressed.bind(i))
		_hand_area.add_child(btn)
		_hand_buttons.append(btn)
	_layout_hand()
	if _pending_card_index >= 0:
		_set_targeting_mode(true)
```

变更点：
- `_hand_container.add_child(btn)` → `_hand_area.add_child(btn)`
- 新增 `btn.size = Vector2(110, 150)`（绝对定位需要显式设置 size）
- `_rebuild_hand()` 末尾调用 `_layout_hand()`（替代 HBoxContainer 的自动布局）

- [ ] **Step 4: 新增 _layout_hand() 函数**

在 `_rebuild_hand()` 函数**之后**添加新函数：

```gdscript
func _layout_hand() -> void:
	var n := _hand_buttons.size()
	if n == 0:
		return
	const CARD_W := 110.0
	const CARD_H := 150.0
	const CONTAINER_H := 170.0
	const MIN_GAP := 8.0
	var W := _hand_area.size.x
	if W == 0.0:
		W = get_viewport_rect().size.x
	var step := 0.0
	if n > 1:
		var natural_step := CARD_W + MIN_GAP
		var max_step := (W - CARD_W) / float(n - 1)
		step = min(natural_step, max_step)
	var total_w := CARD_W + float(n - 1) * step
	var start_x := (W - total_w) / 2.0
	var start_y := (CONTAINER_H - CARD_H) / 2.0
	for i in n:
		_hand_buttons[i].position = Vector2(start_x + float(i) * step, start_y)
```
