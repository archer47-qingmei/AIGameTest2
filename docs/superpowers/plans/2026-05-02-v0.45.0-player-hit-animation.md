# v0.45.0 玩家受击动画 + 战斗界面布局重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

## Task 1: CombatEngine — player_damaged 信号

**Files:**
- Modify: `CombatEngine.gd`

背景：当前 `CombatEngine.gd` 已有 `damage_dealt` 信号用于敌人受伤动画（第 8 行）。本任务新增对称的 `player_damaged` 信号，并在两处玩家扣血逻辑中 emit。

### 当前相关代码

```gdscript
# 第 6-8 行
signal state_changed
signal combat_ended(result: String)
signal damage_dealt(enemy_index: int, amount: int)
```

```gdscript
# end_turn() 第 89-96 行
func end_turn() -> void:
	var venom_count: int = 0
	for card: CardData in hand:
		if card.is_venom:
			venom_count += 1
	if venom_count > 0:
		player.hp = max(0, player.hp - venom_count)
		state_changed.emit()
		if _check_end():
			return
```

```gdscript
# _do_enemy_turn() 第 187-204 行
func _do_enemy_turn() -> void:
	for i in enemies.size():
		if enemies[i].hp <= 0:
			continue
		enemies[i].block = 0
		enemies[i].vulnerable = max(0, enemies[i].vulnerable - 1)
		var action: EnemyActionData = get_enemy_action(i)
		match action.type:
			"attack":
				EffectResolver.apply_damage(enemies[i], player, action.value)
				enemies[i].weak = max(0, enemies[i].weak - 1)
			"poison":
				var venom_card: CardData = load("res://data/cards/venom.tres") as CardData
				for j in action.value:
					_draw_pile.append(venom_card.duplicate())
				_draw_pile.shuffle()
			_:
				enemies[i].add_block(action.value)
```

- [ ] **Step 1: 新增 player_damaged 信号声明**

用 Edit 工具将：
```gdscript

```
替换为：
```gdscript
signal state_changed
signal combat_ended(result: String)
signal damage_dealt(enemy_index: int, amount: int)
signal player_damaged(amount: int)
```

- [ ] **Step 2: end_turn() 毒液扣血后 emit**

用 Edit 工具将：
```gdscript
	if venom_count > 0:
		player.hp = max(0, player.hp - venom_count)
		state_changed.emit()
		if _check_end():
			return
```
替换为：
```gdscript
	if venom_count > 0:
		player.hp = max(0, player.hp - venom_count)
		player_damaged.emit(venom_count)
		state_changed.emit()
		if _check_end():
			return
```

- [ ] **Step 3: _do_enemy_turn() 攻击后 emit**

用 Edit 工具将：
```gdscript
		match action.type:
			"attack":
				EffectResolver.apply_damage(enemies[i], player, action.value)
				enemies[i].weak = max(0, enemies[i].weak - 1)
```
替换为：
```gdscript
		match action.type:
			"attack":
				var hp_before: int = player.hp
				EffectResolver.apply_damage(enemies[i], player, action.value)
				var dmg: int = hp_before - player.hp
				if dmg > 0:
					player_damaged.emit(dmg)
				enemies[i].weak = max(0, enemies[i].weak - 1)
```

```gdscript
func _on_player_damaged(amount: int) -> void:
	_right_panel.pivot_offset = _right_panel.size / 2

	var flash_tween: Tween = create_tween()
	flash_tween.tween_property(_right_panel, "modulate", Color(1.0, 0.2, 0.2), 0.05)
	flash_tween.tween_property(_right_panel, "modulate", Color(1.0, 1.0, 1.0), 0.2)

	var shake_tween: Tween = create_tween()
	shake_tween.tween_property(_right_panel, "scale", Vector2(1.05, 0.95), 0.06)
	shake_tween.tween_property(_right_panel, "scale", Vector2(0.97, 1.03), 0.06)
	shake_tween.tween_property(_right_panel, "scale", Vector2(1.02, 0.99), 0.06)
	shake_tween.tween_property(_right_panel, "scale", Vector2(1.0, 1.0), 0.07)

	var lbl: Label = Label.new()
	lbl.text = "-%d" % amount
	lbl.add_theme_font_size_override("font_size", 24)
	lbl.pivot_offset = Vector2(15, 12)
	lbl.z_index = 10
	add_child(lbl)
	lbl.global_position = _right_panel.get_global_rect().get_center() - Vector2(15, 12)

	var direction: float = 1.0 if randi() % 2 == 0 else -1.0
	var drift_x: float = direction * randf_range(20.0, 60.0)

	var flyout_tween: Tween = create_tween().set_parallel(true)
	flyout_tween.tween_property(lbl, "scale", Vector2(1.5, 1.5), 0.15).set_ease(Tween.EASE_OUT)
	flyout_tween.tween_property(lbl, "position:x", lbl.position.x + drift_x, 0.4)
	flyout_tween.tween_property(lbl, "modulate:a", 0.0, 0.3).set_delay(0.1)
	flyout_tween.finished.connect(lbl.queue_free)

func _on_proceed() -> void:
	GameManager.end_combat(_engine.player.hp)
```
