# v0.68.0 四件新遗物 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增四件遗物（丹心镜、无名修士的护符、碎镜片、不散执念），引入 HEAL_ONCE_IF_HALF_HP 和 PREVENT_DEATH_ONCE 两个新 EffectType，并在 CardPool.RELICS 注册。

**Architecture:** 丹心镜只需新 .tres 文件（复用 DRAW 效果）；无名修士的护符引入一次性触发机制（RelicData 新增 used 字段 + RelicEngine 新增 case）；碎镜片和不散执念使用相同的 PREVENT_DEATH_ONCE 机制，在 CombatEngine._check_end() 死亡判断处拦截，而非走 trigger 系统。三件一次性遗物的 used=true 均在整局首次触发后永久保持，不跨战斗重置（整局一次语义）。

**Tech Stack:** GDScript 4.4，Godot .tres Resource 文件，RelicData/RelicEngine/CombatEngine/CardPool

### Task 3: CombatEngine.gd 死亡拦截

在 `_check_end()` 的玩家死亡检测处，拦截 PREVENT_DEATH_ONCE 遗物。

**Files:**
- Modify: `CombatEngine.gd:397-406`

- [ ] **Step 1: 修改 _check_end()**

当前代码（`CombatEngine.gd` 约第 403 行）：
```gdscript
	if player.hp <= 0:
		combat_ended.emit("game_over")
		return true
```

改为：
```gdscript
	if player.hp <= 0:
		for r: RelicData in _relics:
			if r.effect_type == RelicData.EffectType.PREVENT_DEATH_ONCE and not r.used:
				player.hp = 1
				r.used = true
				return false
		combat_ended.emit("game_over")
		return true
```
