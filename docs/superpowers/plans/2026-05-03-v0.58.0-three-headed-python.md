Replace with:
```gdscript
func _resolve_enemy_actions() -> void:
	_pending_actions.resize(enemies.size())
	for i in enemies.size():
		if enemies[i].hp <= 0:
			_pending_actions[i] = null
			continue
		var data: EnemyData = _enemy_data_list[i]
		var use_phase2: bool = (
			data.phase2_threshold > 0.0
			and not data.phase2_actions.is_empty()
			and float(enemies[i].hp) / float(data.hp) <= data.phase2_threshold
		)
		var action_list: Array[EnemyActionData] = data.phase2_actions if use_phase2 else data.actions
		var action: EnemyActionData
		if data.random_actions:
			action = _weighted_random_action(action_list)
		else:
			action = action_list[(turn_number - 1) % action_list.size()]
		if enemies[i].is_charging and action.type in ["attack", "attack_weak", "attack_vulnerable", "multi_attack"]:
			var doubled := EnemyActionData.new()
			doubled.type = action.type
			doubled.value = action.value * 2
			doubled.count = action.count
			_pending_actions[i] = doubled
		else:
			_pending_actions[i] = action
```

## 本版本内容

新增第二只精英怪：三首妖蟒，引入 phase2 HP 阈值机制。

### 新怪物

**三首妖蟒（Three-Headed Python）**
- HP：147
- 3 回合顺序循环：攻击13 → 格挡14 → 投毒4层
- HP<50% 切换 phase2：攻击13 → 格挡14 → 投毒4层+心魔×2

### 新机制

**phase2 HP 阈值（EnemyData）**
- 新增 `phase2_threshold: float`（0.0=不启用）和 `phase2_actions`
- HP 降至阈值以下后永久切换，顺序计数不重置
- 现有怪物默认 threshold=0，行为不变

**poison_curse（投毒+心魔）**
- value 层毒进抽牌堆，count 张心魔进弃牌堆
- 意图显示："意图：投毒 N + 心魔 ×M"

### 其他改动

- MapGenerator 精英池扩为数组，随机选夺舍邪修或三首妖蟒
- TestSelectScreen 同步显示 13 个敌人组
