## Task 1: 单选卡牌逻辑

**Files:**
- Modify: `combat/CombatScreen.gd`（`_on_card_pressed` 第 58-68 行，`_set_targeting_mode` 第 76-83 行）

### 背景

当前 `_set_targeting_mode(true)` 会把所有手牌按钮设为 `disabled = true`，导致进入瞄准模式后无法点击其他卡牌。用户目前无法取消选中或改选——需要把两处逻辑都修掉。

- [ ] **Step 1: 修改 `_on_card_pressed`**

用 Edit 工具将以下旧代码：

```gdscript
func _on_card_pressed(card_index: int) -> void:
	if _pending_card_index >= 0:
		_pending_card_index = -1
		_set_targeting_mode(false)
		return
	var card: CardData = _engine.hand[card_index]
	if card.target_type in ["none", "all"]:
		_engine.play_card(card_index, -1)
	else:
		_pending_card_index = card_index
		_set_targeting_mode(true)
```

替换为：

```gdscript
func _on_card_pressed(card_index: int) -> void:
	if _pending_card_index >= 0:
		_pending_card_index = -1
		_set_targeting_mode(false)
	var card: CardData = _engine.hand[card_index]
	if card.target_type in ["none", "all"]:
		_engine.play_card(card_index, -1)
	else:
		_pending_card_index = card_index
		_set_targeting_mode(true)
```

（只删掉了 `return` 一行）

- [ ] **Step 2: 修改 `_set_targeting_mode`**

用 Edit 工具将以下旧代码：

```gdscript
func _set_targeting_mode(active: bool) -> void:
	_btn_end_turn.disabled = active
	for btn: Button in _hand_buttons:
		btn.disabled = active
	for i in _enemies_container.get_child_count():
		var btn: Button = _enemies_container.get_child(i) as Button
		if i < _engine.enemies.size() and _engine.enemies[i].hp > 0:
			btn.disabled = not active
```

替换为：

```gdscript
func _set_targeting_mode(active: bool) -> void:
	_btn_end_turn.disabled = active
	for i in _enemies_container.get_child_count():
		var btn: Button = _enemies_container.get_child(i) as Button
		if i < _engine.enemies.size() and _engine.enemies[i].hp > 0:
			btn.disabled = not active
```

（删掉了手牌按钮的 disabled 循环——手牌按钮始终可点击）

- [ ] **Step 3: 提交**

```bash
git add combat/CombatScreen.gd
git commit -m "feat: single-select card — clicking new card replaces selection"
```

## Task 2: 卡牌固定尺寸

**Files:**
- Modify: `combat/CombatScreen.gd`（`_rebuild_hand` 第 119-131 行）
- Modify: `combat/CombatScreen.tscn`（HandScroll 节点第 45-47 行）

- [ ] **Step 1: 修改 `_rebuild_hand` 中的 Button 创建**

用 Edit 工具将以下旧代码：

```gdscript
	for i in _engine.hand.size():
		var card: CardData = _engine.hand[i]
		var btn: Button = Button.new()
		btn.text = card.get_description()
		btn.pressed.connect(_on_card_pressed.bind(i))
		_hand_container.add_child(btn)
		_hand_buttons.append(btn)
```

替换为：

```gdscript
	for i in _engine.hand.size():
		var card: CardData = _engine.hand[i]
		var btn: Button = Button.new()
		btn.custom_minimum_size = Vector2(110, 150)
		btn.autowrap_mode = TextServer.AUTOWRAP_WORD_SMART
		btn.vertical_alignment = VERTICAL_ALIGNMENT_TOP
		btn.text = card.get_description()
		btn.pressed.connect(_on_card_pressed.bind(i))
		_hand_container.add_child(btn)
		_hand_buttons.append(btn)
```

- [ ] **Step 2: 修改 CombatScreen.tscn 中 HandScroll 高度**

用 Edit 工具将：

```
[node name="HandScroll" type="ScrollContainer" parent="VBoxContainer"]
custom_minimum_size = Vector2(0, 80)
```

替换为：

```
[node name="HandScroll" type="ScrollContainer" parent="VBoxContainer"]
custom_minimum_size = Vector2(0, 170)
```
