> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增精英怪夺舍邪修，引入 steal_block 夺舍机制。

**Architecture:** 分三层：①Engine/Screen 扩展 steal_block action type；②夺舍邪修 .tres 数据文件；③MapGenerator 替换精英占位路径 + 版本号。steal_block 在 `_do_enemy_turn()` 中直接操作 player.block，偷光玩家当前格挡并转移给敌人；若玩家格挡为 0 则敌人改为 +3 力量。

**Tech Stack:** GDScript 4，Godot 4.4 .tres resource 文件

### Task 1: Engine + Screen 扩展 steal_block

**Files:**
- Modify: `CombatEngine.gd`
- Modify: `combat/CombatScreen.gd`

- [ ] **Step 1: 修改 `CombatEngine.gd` — `_do_enemy_turn()` 新增 steal_block 分支**

在 `"pre_charge":` 分支之后、`"group_strengthen":` 之前插入：

```gdscript
			"steal_block":
				var stolen: int = player.block
				player.block = 0
				if stolen > 0:
					enemies[i].add_block(stolen)
				else:
					enemies[i].strength += 3
```

插入后该区域完整内容：

```gdscript
			"charge":
				enemies[i].is_charging = true
			"pre_charge":
				pass
			"steal_block":
				var stolen: int = player.block
				player.block = 0
				if stolen > 0:
					enemies[i].add_block(stolen)
				else:
					enemies[i].strength += 3
			"group_strengthen":
				for j in enemies.size():
					if enemies[j].hp > 0:
						enemies[j].strength += action.value
```

- [ ] **Step 2: 修改 `combat/CombatScreen.gd` — `_intent_text()` 新增 steal_block case**

在 `"charge":` case 之前插入：

```gdscript

			return "意图：夺舍"
```

插入后该区域完整内容：

```gdscript
		"steal_block":
			return "意图：夺舍"
		"charge":
			return "意图：蓄力"
		"pre_charge":
			return "意图：蓄力"
```

### Task 2: 夺舍邪修数据文件

HP = 137（范围 130-145 中值），4 回合顺序循环：攻击9 → 防御10 → 夺舍 → 连斩7×2

**Files:**
- Create: `data/enemies/elite_sorcerer.tres`
- Create: `data/enemy_groups/single_elite_sorcerer.tres`

- [ ] **Step 1: 创建 `data/enemies/elite_sorcerer.tres`**

load_steps=7：2 ext_resource + 4 sub_resource + 1 resource

```
[gd_resource type="Resource" script_class="EnemyData" load_steps=7 format=3]

[ext_resource type="Script" path="res://EnemyData.gd" id="1_EnemyData"]
[ext_resource type="Script" path="res://EnemyActionData.gd" id="2_EnemyActionData"]

[sub_resource type="Resource" id="Action_attack"]
script = ExtResource("2_EnemyActionData")
type = "attack"
value = 9

[sub_resource type="Resource" id="Action_block"]
script = ExtResource("2_EnemyActionData")
type = "block"
value = 10

[sub_resource type="Resource" id="Action_steal"]
script = ExtResource("2_EnemyActionData")
type = "steal_block"
value = 0

[sub_resource type="Resource" id="Action_multi"]
script = ExtResource("2_EnemyActionData")
type = "multi_attack"
value = 7
count = 2

[resource]
script = ExtResource("1_EnemyData")
display_name = "夺舍邪修"
hp = 137
actions = Array[ExtResource("2_EnemyActionData")]([SubResource("Action_attack"), SubResource("Action_block"), SubResource("Action_steal"), SubResource("Action_multi")])
```

- [ ] **Step 2: 创建 `data/enemy_groups/single_elite_sorcerer.tres`**

```
[gd_resource type="Resource" script_class="EnemyGroupData" load_steps=4 format=3]

[ext_resource type="Script" path="res://EnemyGroupData.gd" id="1_GroupData"]

[ext_resource type="Resource" path="res://data/enemies/elite_sorcerer.tres" id="3_elite_sorcerer"]

[resource]
```
