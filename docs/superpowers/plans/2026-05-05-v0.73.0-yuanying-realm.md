# v0.73.0 元婴境 + 意图详情 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 扩展游戏到元婴境（第三境界），添加点击意图查看详情功能，并实现支撑新怪物行为的引擎机制。

**Architecture:** 在 EnemyActionData 加 description 字段，CombatScreen 的 BtnOverlay 从 disabled 改为可点击显示 InfoPanel；引擎扩展 8 种新 action type、skill_locked/turns_alive/energy_penalty 三个新状态；MapGenerator 加 realm=2 分支。

**Tech Stack:** Godot 4.4 GDScript，无自动测试框架，所有验证通过游戏内测试战斗手动确认。

**Files:**
- Modify: `EnemyActionData.gd`
- Modify: `combat/CombatScreen.gd:76-127` (`_build_enemy_panels`)、`110-127` (`_refresh_ui` enemy block)、新增 `_on_intent_pressed`

- [ ] **Step 1: 给 EnemyActionData 加 description 字段**

将 `EnemyActionData.gd` 完整替换为：

```gdscript
class_name EnemyActionData
extends Resource

@export var type: String
@export var value: int
@export var weight: int = 1
@export var count: int = 1
@export var display_label: String = ""

```

- [ ] **Step 2: 修改 CombatScreen._build_enemy_panels 启用 BtnOverlay**

找到 `_build_enemy_panels()` 函数中的这段代码（约第 98-104 行）：

```gdscript
		var btn := Button.new()
		btn.name = "BtnOverlay"
		btn.flat = true
		btn.disabled = true
		panel.add_child(btn)
```

替换为：

```gdscript
		var btn := Button.new()
		btn.name = "BtnOverlay"
		btn.flat = true
		btn.disabled = true
		btn.pressed.connect(_on_intent_pressed.bind(i))
		panel.add_child(btn)
```

- [ ] **Step 3: 修改 CombatScreen._refresh_ui 中对 btn 的处理**

找到 `_refresh_ui()` 中的这段（每个敌人的 if/else 块内两处 `btn.disabled = true`）：

```gdscript
		if e.hp <= 0:
			lbl_info.text = "%s\n(已死亡)" % e.display_name
			lbl_intent.text = ""
			btn.disabled = true
		else:
			var action: EnemyActionData = _engine.get_enemy_action(i)
			lbl_info.text = "%s\nHP:%d/%d 挡:%d" % [e.display_name, e.hp, e.max_hp, e.block]
			lbl_intent.text = _intent_text(action, e)
			btn.disabled = true
```

替换为：

```gdscript
		if e.hp <= 0:
			lbl_info.text = "%s\n(已死亡)" % e.display_name
			lbl_intent.text = ""
			btn.disabled = true
		else:
			var action: EnemyActionData = _engine.get_enemy_action(i)
			lbl_info.text = "%s\nHP:%d/%d 挡:%d" % [e.display_name, e.hp, e.max_hp, e.block]
			lbl_intent.text = _intent_text(action, e)
			btn.disabled = (action == null or action.description == "")
```

- [ ] **Step 4: 在 CombatScreen 末尾（_on_proceed 之前）新增 _on_intent_pressed 方法**

```gdscript
func _on_intent_pressed(enemy_index: int) -> void:
	var action: EnemyActionData = _engine.get_enemy_action(enemy_index)
	if action == null or action.description == "":
		return
	var title: String = action.display_label if action.display_label != "" else "意图"
	_info_panel.show_info(title, action.description)
```

- [ ] **Step 5: 验证**

运行游戏 → 测试战斗 → 选任意敌人（description 暂为空，btn 应 disabled）→ 确认无崩溃。后续 Task 2/3 填入 description 后再验证点击效果。

```bash

```
