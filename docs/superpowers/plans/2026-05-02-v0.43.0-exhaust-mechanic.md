# v0.43.0 功法 Exhaust 机制 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 功法类卡牌打出后进入消耗区，不进入弃牌堆，不会被重洗入抽牌堆；DeckViewPanel 新增「消耗区」标签展示已消耗的牌。

**Architecture:** `CombatEngine` 新增 `_exhaust_pile`，`play_card()` 按 `card.card_type` 分流；`CombatScreen` 新增 `_exhaust_list` 节点引用并在牌组查看时填充。改动完全在现有架构内，不新增模块。

**Tech Stack:** Godot 4 GDScript，TabContainer。

## Task 1: CombatEngine exhaust pile

**Files:**
- Modify: `CombatEngine.gd`

背景：当前 `play_card()` 末尾 `_discard_pile.append(card)` 将所有打出的牌都放入弃牌堆。本任务新增消耗区字段，并对 `card_type == "功法"` 的牌改为放入消耗区。

相关现有代码（供参考）：
```gdscript
# CombatEngine.gd 第 17-18 行
var _draw_pile: Array[CardData] = []
var _discard_pile: Array[CardData] = []

# play_card() 第 67-68 行
hand.remove_at(card_index)
_discard_pile.append(card)

# get_discard_pile() 第 51-52 行
func get_discard_pile() -> Array[CardData]:
    return _discard_pile.duplicate()
```

- [ ] **Step 1: 新增 `_exhaust_pile` 字段**

用 Edit 工具将：
```gdscript
var _draw_pile: Array[CardData] = []
var _discard_pile: Array[CardData] = []
```
替换为：
```gdscript
var _draw_pile: Array[CardData] = []
var _discard_pile: Array[CardData] = []
var _exhaust_pile: Array[CardData] = []
```

- [ ] **Step 2: 修改 `play_card()` 分流逻辑**

用 Edit 工具将：
```gdscript
	hand.remove_at(card_index)
	_discard_pile.append(card)
```
替换为：
```gdscript
	hand.remove_at(card_index)
	if card.card_type == "功法":
		_exhaust_pile.append(card)
	else:
		_discard_pile.append(card)
```

- [ ] **Step 3: 新增 `get_exhaust_pile()` 方法**

用 Edit 工具将：
```gdscript
func get_discard_pile() -> Array[CardData]:
	return _discard_pile.duplicate()
```
替换为：
```gdscript
func get_discard_pile() -> Array[CardData]:
	return _discard_pile.duplicate()

func get_exhaust_pile() -> Array[CardData]:
	return _exhaust_pile.duplicate()
```

- [ ] **Step 4: 提交**

```bash
git add CombatEngine.gd
git commit -m "feat: exhaust 功法 cards instead of discarding"
```

## Task 3: CombatScreen.gd 接入消耗区

**Files:**
- Modify: `combat/CombatScreen.gd`

背景：需新增 `_exhaust_list` 的 `@onready` 引用，并在 `_on_view_deck_pressed()` 中填充消耗区列表。同时将消耗区的牌纳入「完整牌组」视图。

当前 `_on_view_deck_pressed()`：
```gdscript
func _on_view_deck_pressed() -> void:
	var all_cards: Array[CardData] = []
	for card: CardData in _engine.hand:
		all_cards.append(card)
	for card: CardData in _engine.get_draw_pile():
		all_cards.append(card)
	for card: CardData in _engine.get_discard_pile():
		all_cards.append(card)
	_populate_list(_all_cards_list, all_cards)
	_populate_list(_draw_list, _engine.get_draw_pile())
	_populate_list(_discard_list, _engine.get_discard_pile())
	_deck_view_panel.show()
```

- [ ] **Step 1: 新增 `_exhaust_list` 的 `@onready` 引用**

用 Edit 工具将：
```gdscript
@onready var _discard_list: VBoxContainer   = $DeckViewPanel/VBoxContainer/TabContainer/弃牌堆/DiscardList
```
替换为：
```gdscript
@onready var _discard_list: VBoxContainer   = $DeckViewPanel/VBoxContainer/TabContainer/弃牌堆/DiscardList
@onready var _exhaust_list: VBoxContainer   = $DeckViewPanel/VBoxContainer/TabContainer/消耗区/ExhaustList
```

- [ ] **Step 2: 更新 `_on_view_deck_pressed()`**

用 Edit 工具将整个方法替换为：
```gdscript
func _on_view_deck_pressed() -> void:
	var all_cards: Array[CardData] = []
	for card: CardData in _engine.hand:
		all_cards.append(card)
	for card: CardData in _engine.get_draw_pile():
		all_cards.append(card)
	for card: CardData in _engine.get_discard_pile():
		all_cards.append(card)
	for card: CardData in _engine.get_exhaust_pile():
		all_cards.append(card)
	_populate_list(_all_cards_list, all_cards)
	_populate_list(_draw_list, _engine.get_draw_pile())
	_populate_list(_discard_list, _engine.get_discard_pile())
	_populate_list(_exhaust_list, _engine.get_exhaust_pile())
	_deck_view_panel.show()
```

- [ ] **Step 3: 提交**

```bash

git commit -m "feat: show exhaust pile in DeckViewPanel"
```
