# v0.62.0 拖牌体验优化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 拖动卡牌时卡牌上移半张高度避开手指遮挡，同时在卡牌文字中以绿/红/默认色实时显示对当前目标的真实伤害值。

**Architecture:** 三层修改：CardData 新增 `get_description_with_dmg()` 生成含 BBCode 颜色标签的描述字符串（real > base → 绿，real < base → 红，相等无色）；DragLayer ghost card 的 Label 改为 RichTextLabel 以支持 BBCode，修正拖拽偏移并新增 `update_card_description()`；CombatScreen 协调两者，在拖拽开始和目标切换时推送实时描述文字。CardData 不引用 Combatant，伤害计算由 CombatScreen 通过 EffectResolver.preview_damage() 完成。

**Tech Stack:** GDScript 4、Godot 4.4

- [ ] **Step 1: 读取文件，确认 `_format_description` 的签名和 damage 块位置**

- [ ] **Step 2: 修改 `_format_description`，加可选 `real_dmg: int = -1` 参数，生成 BBCode 颜色**

  只修改函数签名和 `if v.dmg > 0:` 块，其余行不动：

  ```gdscript
  func _format_description(name: String, show_bonuses: bool, real_dmg: int = -1) -> String:
      var v: Dictionary = _collect_effects()
      var desc: String = "%s\n费用:%d" % [name, cost]
      if v.dmg > 0:
          var dmg_str: String
          if real_dmg >= 0:
              if real_dmg > v.dmg:
                  dmg_str = "[color=green]%d[/color]" % real_dmg
              elif real_dmg < v.dmg:
                  dmg_str = "[color=red]%d[/color]" % real_dmg
              else:
                  dmg_str = "%d" % real_dmg
          else:
              dmg_str = _fmt(v.dmg, v.dmg_bonus, show_bonuses)
          if v.dmg_times > 1:
              desc += "  攻:%s×%d" % [dmg_str, v.dmg_times]
          else:
              desc += "  攻:%s" % dmg_str
          if sword_intent_consume_bonus > 0:
              desc += "(+剑意×%d)" % sword_intent_consume_bonus
      # blk/drw/nrg/... 以下所有行保持原样不变
  ```

  颜色规则：real_dmg > base（v.dmg）→ 绿色；real_dmg < base → 红色；相等 → 无色。
  BBCode 只在 `real_dmg >= 0` 时生成，普通 `get_description()` 调用（real_dmg = -1）不受影响。

- [ ] **Step 3: 新增 `get_description_with_dmg()` 方法**

          return "%s\n费用:%d  %s" % [card_name, cost, special_text]
      return _format_description(card_name, false, real_dmg)

- [ ] **Step 1: 读取文件，定位 `_create_ghost` 和 `_on_move`**

  Read `E:\MyWork\AIGameTest2\combat\DragLayer.gd`。找到：
  - `_create_ghost` 中的 `var lbl := Label.new()` 和 `add_child(lbl)`
  - `_on_move` 中的 `_ghost_card.position = local_pos - GHOST_SIZE / 2.0`

- [ ] **Step 2: `_create_ghost` 中 Label → RichTextLabel**

  将 `_create_ghost` 函数内的 Label 替换为 RichTextLabel：

  ```gdscript
  func _create_ghost(text: String, local_pos: Vector2) -> void:
      _ghost_card = Panel.new()
      _ghost_card.custom_minimum_size = GHOST_SIZE
      _ghost_card.size = GHOST_SIZE
      var style := StyleBoxFlat.new()
      style.bg_color = Color(0.2, 0.2, 0.2)
      style.set_border_width_all(2)
      style.border_color = Color.WHITE
      _ghost_card.add_theme_stylebox_override("panel", style)
      _ghost_card.modulate.a = 0.9
      _ghost_card.mouse_filter = MOUSE_FILTER_IGNORE
      var lbl := RichTextLabel.new()   # ← 原来是 Label.new()
      lbl.bbcode_enabled = true        # ← 新增
      lbl.text = text
      lbl.autowrap_mode = TextServer.AUTOWRAP_WORD_SMART
      lbl.scroll_active = false        # ← 新增，禁用滚动条
      lbl.anchor_right = 1.0
      lbl.anchor_bottom = 1.0
      lbl.mouse_filter = MOUSE_FILTER_IGNORE
      _ghost_card.add_child(lbl)
      _ghost_card.position = local_pos
      add_child(_ghost_card)
  ```

  注意：`vertical_alignment` 在 RichTextLabel 中不适用，删去即可（RichTextLabel 默认从顶部开始）。

- [ ] **Step 3: 修正 `_on_move` 拖拽偏移**
