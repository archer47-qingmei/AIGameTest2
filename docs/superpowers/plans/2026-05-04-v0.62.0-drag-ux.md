# v0.62.0 拖牌体验优化 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 拖动卡牌时卡牌上移半张高度避开手指遮挡，同时在卡牌文字中实时显示对当前目标的真实伤害值。

**Architecture:** 三层修改：CardData 新增 `get_description_with_dmg()` 接受预计算的真实伤害整数；DragLayer 修正拖拽偏移并新增 `update_card_description()` 方法；CombatScreen 协调两者，在拖拽开始和目标切换时向 DragLayer 推送实时描述文字。CardData 不引用 Combatant（数据层约束），伤害计算由 CombatScreen 通过 EffectResolver.preview_damage() 完成。

**Tech Stack:** GDScript 4、Godot 4.4

---

| 文件 | 变更 |

      var desc: String = "%s\n费用:%d" % [name, cost]

          var dmg_str: String

- `_ghost_card` 的唯一子节点是 Label（`_create_ghost` 中 `add_child(lbl)` 仅一次）

- [ ] **Step 1: 读取文件，定位 `_on_move` 中的偏移行**

  Read `E:\MyWork\AIGameTest2\combat\DragLayer.gd`。找到：
  ```gdscript

  ```

- [ ] **Step 2: 修正 `_on_move` 拖拽偏移**

  将上面那行改为：
  ```gdscript
  _ghost_card.position = local_pos - Vector2(GHOST_SIZE.x / 2.0, GHOST_SIZE.y)
  ```

  结果：卡牌水平居中于手指，底边比手指坐标高 75px（即卡牌中心高手指 75px）。

- [ ] **Step 3: 新增 `update_card_description()` 方法**

  在文件末尾 `_draw()` 之后追加：

  ```gdscript
  func update_card_description(text: String) -> void:
      if _ghost_card == null:
          return
      var lbl := _ghost_card.get_child(0) as Label
      if lbl != null:
          lbl.text = text
  ```

  `get_child(0)` 安全原因：`_create_ghost` 中只 `add_child` 了一个 Label，且 `_cleanup` 调用 `_ghost_card.queue_free()` 前已有 `if _ghost_card == null` 守卫。
