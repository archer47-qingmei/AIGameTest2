# v0.20.0 架构清理 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 修复三处违反架构原则的技术债：NodeData 拆分为 NodeConfig（Resource）+ NodeData（RefCounted）、治疗逻辑移入 PlayerState、CombatScreen 不再直接写 player_state。

**Tech Stack:** Godot 4.4 GDScript 严格类型，RefCounted 逻辑层，Resource 数据层，Control UI 层。Godot 可执行文件路径：`D:/steam/steamapps/common/Godot Engine/godot.windows.opt.tools.64.exe`，项目路径：`E:/MyWork/AIGameTest2`。

```gdscript
extends SceneTree

func _initialize() -> void:
	DirAccess.make_dir_recursive_absolute("res://data/nodes")

	var configs: Array[Dictionary] = [
		{"name": "node_a1", "type": NodeConfig.Type.COMBAT, "enemy": "res://data/enemies/jaw_worm.tres", "col": 0, "row": 0},
		{"name": "node_a2", "type": NodeConfig.Type.COMBAT, "enemy": "res://data/enemies/fire_lizard.tres", "col": 0, "row": 1},
		{"name": "node_b1", "type": NodeConfig.Type.REST,   "enemy": "",                                    "col": 1, "row": 0},
		{"name": "node_b2", "type": NodeConfig.Type.COMBAT, "enemy": "res://data/enemies/fire_lizard.tres", "col": 1, "row": 1},
		{"name": "node_c1", "type": NodeConfig.Type.COMBAT, "enemy": "res://data/enemies/boss.tres",        "col": 2, "row": 0},
	]

	for c: Dictionary in configs:
		var cfg: NodeConfig = NodeConfig.new()
		cfg.type = c["type"]
		if c["enemy"] != "":
			cfg.enemy_data = load(c["enemy"]) as EnemyData
		cfg.column = c["col"]
		cfg.row = c["row"]
		var path: String = "res://data/nodes/%s.tres" % c["name"]
		ResourceSaver.save(cfg, path)
		print("Saved: ", path)

	quit()
```

### Task 4: PlayerState 增加 apply_rest_heal()

**Files:**
- Modify: `PlayerState.gd`

- [ ] **Step 1**: 替换 `PlayerState.gd` 完整内容：

```gdscript
class_name PlayerState
extends RefCounted

var deck: Array[CardData] = []
var hp: int = 70
var max_hp: int = 70
var map_all_nodes: Array[NodeData] = []
var available_nodes: Array[NodeData] = []
var completed_nodes: Array[NodeData] = []
var current_node: NodeData = null
var last_rest_heal: int = 0

func apply_rest_heal() -> int:
	last_rest_heal = int(max_hp * 0.3)
	hp = mini(hp + last_rest_heal, max_hp)
	return last_rest_heal
```
